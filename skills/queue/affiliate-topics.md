# affiliate-topics.md ── アフィリエイト商品候補リスト
next-topics.mdのアフィリエイト版。投稿したい商品をここで管理する。
/affiliate-writer はここから商品を選んで投稿を生成する。

> ステータスの流れ：候補 → 執筆中 → affiliate-queue.md済み

## 楽天イベントカレンダー（成約率UP期間）

| イベント | 時期 | 対応 |
|---------|------|------|
| お買い物マラソン | 月1〜2回（不定期） | 前日〜初日に投稿優先 |
| 楽天スーパーSALE | 3月・6月・9月・12月 | SALE開始日に集中投稿 |
| 5と0のつく日（ポイント5倍） | 毎月5・10・15・20・25・30日 | 当日朝に投稿 |
| 39ショップの日 | 毎月9・19・29日 | 当日投稿 |

確認先：https://calendar.rakuten.co.jp/

---

## 記入フォーマット

```
## 候補N：[商品カテゴリ／テーマ一言]
**商品名**：
**楽天URL**：https://item.rakuten.co.jp/...
**アフィリエイトURL**：https://a.r10.to/...
**時期**：[通年 / 夏季 / 冬季 / 入園準備 など]
**イベント対応**：[マラソン / スーパーSALE / 5と0 / 値上げ前 / なし]
**投稿アングル**：[どんな育児の悩み・場面から入るか1行で]
**ステータス**：候補
```

> ⚠️ **イベント対応フィールドの値は必ず上記タグ名を使う**
> event-prep.py / event-sort.py がこの値を読み取るため、表記ゆれがあると検出されない。
> 複数イベントに対応する場合は「マラソン / スーパーSALE」のようにスラッシュ区切りで記入。
