## 図解投稿の運用ルール

### 頻度・配置

- **週1本**を目安にpost-queue.mdに組み込む
- **土曜日の枠**が最適（週末にスマホを見るフォロワーの保存率が高い）
- アフィリエイト投稿の**2〜3本前**に配置する（図解で信頼↑→アフィリエイト成約↑）
- 図解投稿を連続で入れない（週1本ルール）

### コンテンツ選定の優先順位

1. **保育士専門知識系**（月齢別・発達・離乳食・睡眠）→ 保存率が最も高い
2. **育児グッズ比較・選び方系** → アフィリエイトとの相性が良い
3. **共感・あるある系リスト** → コメント・シェアが取れる

### post-queue.mdへの追記フォーマット

```
## 投稿[番号]
**テーマ**：「[図解タイトル]」
**カテゴリ**：専門知識（B）／図解投稿
**時期**：[通年 or 季節]
**図解タイプ**：[リスト型 / タイル型 など]

**本文**（キャプション）
[1〜3行のキャプション]

**図解生成メモ**
※ /zu-writer で生成したプロンプトをChatGPT/Gemini（思考モード）に貼り付けて画像を作成してから投稿する

---
```

### 効果測定

- 図解投稿のいいね・保存・フォロワー増加をpost-history.mdに記録する
- 「どの図解タイプが伸びたか」を3本溜まった時点で比較する
- 伸びた図解タイプ・テーマを次回の図解選定に活かす

## Threads投稿キャプションのルール（図解投稿用）

カピおじのSTEP 12に準じた、ちゃむ。版のキャプションルール：

- **1〜3行以内**（図解が主役・キャプションは添える程度）
- **インパクトのある言葉をコンパクトに**
- 「届けたい相手に響く言葉」を必ず入れる
- 図解の内容を要約しない（図解を見てもらう）
- 「保存してね」「参考にしてね」はNG → 図解の価値で自然に保存させる

```
例：
「知らないとずっと損する。」（リスト型・育児豆知識）
「これ知ってから、寝かしつけが変わった。」（フロー型）
「全部のママに届いてほしい。」（タイル型・グッズまとめ）
```

## 図解キャプション フック集

キャプション1行目に使う。**図解タイプ・テーマ別に選ぶ。**

### 損失回避・緊急感系
- 「知らないとずっと損する。」
- 「これ知らずに育ててたのか、って思った。」
- 「もっと早く知りたかったやつ。」
- 「うちの子に使う前に知っておけばよかった。」
- 「産前に誰も教えてくれなかった。」
- 「これ見逃してたら、たぶん今も悩んでた。」

### 保育士権威・経験系
- 「16年保育士やって、これが一番聞かれる。」
- 「保育士が本音で言う。」
- 「保育現場で見てきた、正直な話。」
- 「保育士1000人に聞いたら、たぶん全員これ言う。」
- 「現場では当たり前なのに、家庭では知られてない。」

### 共感・あるある系
- 「これ、全員通る道だよね。」
- 「ずっとモヤモヤしてたこと、これだったんだ。」
- 「うちだけじゃなかったんだ、ってなる図解。」
- 「育児でいちばん困ったこと、全部ここにある。」
- 「ワンオペのときに知りたかった。」

### 断言・驚き系
- 「断言する。これが一番大事。」
- 「え、こんなに違うの。」
- 「これがすべて。」
- 「シンプルだけど、誰もやってない。」
- 「複雑に考えすぎてた。答えはこれだけ。」

### 保存誘導系（自然に保存させる）
- 「あとで絶対使う。」
- 「子どもが大きくなるまで手元に置いといて。」
- 「迷ったらここに戻ってきて。」
- 「悩んでるママ全員に届けたい。」

### ターゲット絞り込み系
- 「0〜2歳のいるおうち向け。」
- 「寝かしつけで毎晩消耗してるママへ。」
- 「離乳食で何を食べさせていいか迷ってる人へ。」
- 「保育園入れたてのパパママ、これだけ見て。」

## 図解キャプション CTA集

キャプション末尾、またはコメント欄に置く。**図解投稿はCTAを押しつけない。**

### フォロー誘導（自然な形）
- 「こういう図解、またつくります」
- 「育児の「わからない」を図解にしていくので → フォローしてもらえると続けられる」
- 「月齢別シリーズ、続き出します」

### 保存誘導（直接的にならず）
- 「月齢が上がったときにまた見返してね」
- 「迷ったときの判断基準にして」
- 「印刷して冷蔵庫に貼ってる人もいる（ガチ）」

### コメント引き出し系（クローズド）
- 「うちの子、当てはまってたものあった？」
- 「A派？B派？どっちに近い？」
- 「何番が刺さった？」
- 「これ知ってた？知らなかった？」

### 拡散誘導（自然な形）
- 「まわりのママにも届いてほしい」
- 「パパにも見せてあげて」
- 「プレママにも刺さる内容だと思う」

### 次回予告型
- 「次は〇〇の図解つくります」
- 「続き出すので待っててね」
- 「リクエストあったら教えて」

## NG（図解投稿での禁止事項）

- 「保存してね！」「フォローしてね！」の直接命令 → 自然な言葉に変える
- キャプションに図解の内容を全部書く → 図解を見る理由がなくなる
- 「参考になれば嬉しいです」「ぜひ試してみてください」 → AI文体でNG
- 図解タイトルと同じ文章をキャプションに使う → 重複で印象が薄れる
