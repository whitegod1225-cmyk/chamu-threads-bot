# 06_schedule.md ── 投稿スケジュール
## 投稿本数
- **平日（月〜金）：6本**
- **土日：5本**

## 実際のcronスロット（JST）
| 曜日 | スロット |
|------|---------|
| 平日 | 06:00 / 09:00 / 12:30 / 18:00 / 21:00 / 23:00 |
| 土日 | 07:30 / 11:30 / 15:00 / 19:00 / 22:00 |

## ⚠️ 朝一番スロットのアフィリエイト投稿禁止
- 平日 06:00・土日 07:30 の第1スロットにはアフィリエイト投稿を入れない
- post.py が自動検出して非アフィリ投稿と入れ替えるが、キュー作成時点でも守ること
- アフィリエイト投稿の推奨スロット：21:00 / 23:00（エンゲージメント最大帯）

## 時間帯別コンテンツ傾向
- 6:00：共感系・今日の一言系
- 9:00：育児知識・保育系
- 12:00：グッズ紹介・短文系
- 15:00：悩み系・感情寄り添い系
- 18:00：節約・料理雑貨・共感系
- 21:00：深い悩み・夫婦関係・note誘導
- 0:00：寄り添い・孤独感・短文

## 曜日別推奨スロット
- 平日：6・9・12・18・21
- 土曜：9・12・15・18・21
- 日曜：9・12・15・21・0

## スマホ利用ピーク時間帯（読者行動ベース）

**平日**
| 時間帯 | 理由 |
|--------|------|
| 7:00〜8:00 | 通勤・朝の支度中 |
| 12:00〜13:00 | 昼休みのスマホ時間 |
| 17:30〜18:30 | 退勤〜帰宅タイム |
| **20:00〜21:00** | **最もスマホ利用が多い（バズ投稿はここ）** |
| 22:00〜 | ベッドでスマホタイム |

**休日**
| 時間帯 | 理由 |
|--------|------|
| 9:00〜10:00 | ゆっくり起きる時間 |
| 15:00〜16:00 | 余裕がある人の追加枠 |
| 以降 | 平日と同様 |

**→ バズらせたい投稿・アフィリエイト投稿は21:00スロット（20〜21時台）を優先する**
