# 13_cta-patterns.md ── CTA集（コール・トゥ・アクション）

購入者限定特典「CTA集」をちゃむ。用にknowledge化したもの。
アフィリエイト投稿のコメント欄末尾（URLの直前）に使う。

> **使い方**：コメント欄の最後、URLの1行前に置く。
> ちゃむ。トーンに合わせて「友人への語りかけ」レベルに調整する。
> 「実質タダ」「持ってないの自分だけ説」などFOMO煽りが強いものは使わない。

---

## カジュアル誘導型（ちゃむ。で最もよく使う型）
自然に誘導できる。押しつけ感がなくちゃむ。トーンに最も馴染む。

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 気になる人はこちら | 万能・自然な誘導 |
| リンク貼っとくね | 友達感覚の誘導 |
| ここで買えるよ | 購入導線の提示 |
| チェックしてみて | 軽い推奨 |
| 詳細は下のリンクから | シンプルな案内 |

---

## 背中押し型（迷っている人を後押し）
「見るだけでもいい」「ハードルが低い」形で誘導する。

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| とりあえずページだけでも見て | 低いハードル・クリック促進 |
| 見るだけならタダだし | リスクゼロ訴求 |
| 気になったら覗いてみて | 軽い誘導 |
| 迷ってる人は一回見た方がいい | 決断の後押し |

---

## 体験談型（「うちはここで買った」感）
実体験ベースでちゃむ。らしさが出る。信頼感UP。

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 私はここで買いました | 実体験の信頼性 |
| 買ってから毎日使ってる | 日常使いアピール |
| リピート〇回目だけど飽きない | 継続使用の証明 |
| 友達にも勧めまくってる | 口コミの自然な流れ |

---

## 緊急性訴求型（セール・イベント期間限定で使う）
楽天スーパーSALE・マラソン・5と0のつく日など、イベント時に使う。
**普段使いしすぎると信頼が落ちるので、本当にセール中のときだけ使う。**

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 今だけ半額になってる | セール・タイムセール |
| セール終わる前に見て | 期間限定の強調 |
| 在庫残りわずかっぽい | 品薄状況 |
| 明日には値段戻るかも | 価格変動の可能性 |
| このタイミング逃したら後悔する | 機会損失回避 |

---

## お得感訴求型（コスパ・節約スタイルのちゃむ。と相性◎）

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| コスパやばすぎて笑った | コストパフォーマンス強調 |
| この値段でこのクオリティはバグ | 異常な安さの強調 |
| ポイント爆盛りキャンペーン中 | ポイント還元アピール |
| 1000円以下で買えるやつ | プチプラ訴求 |

---

## 希少性訴求型（早めに知ってほしい系）

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| バズる前に教えとく | 先行者利益 |
| 知ってる人だけ得してる | 情報格差の提示 |
| これ見つけた時即買いした | 発見の喜び |
| 売り切れる前にチェック | 品切れリスク |
| 再入荷したから急いで | 再入荷の希少性 |

---

## 比較優位型（比較・検証後の結論として）

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 高いやつと変わらないクオリティ | コスパの証明 |
| 他の商品試したけど結局これ | 比較後の結論 |
| 同じ値段なら断然こっち | 選択の明確化 |
| ランキング上位の理由が分かる | 人気の裏付け |

---

## 問題解決型（悩み解決の締めに）

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 〇〇で困ってる人に教えたい | ターゲット明確化 |
| もっと早く知りたかった | 後悔の裏返し（共感を引き出す） |
| 同じ悩みの人は見て | 共感からの誘導 |
| これさえあれば大丈夫 | 安心感の提供 |

---

## 数字訴求型（信頼性・実績を示す）

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| レビュー1万件超えの実力 | 社会的証明 |
| ☆4.8の高評価には理由がある | 評価の信頼性 |
| 1年使っても壊れない | 耐久性アピール |
| 〇〇円引きクーポンあり | 具体的割引額 |

## FOMO型（乗り遅れ回避）
**ちゃむ。トーンでは基本使わない。使う場合は最も穏やかなものだけ。**

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| みんな買ってるやつ | 流行への乗り遅れ回避 |
| バズってる理由を確認 | トレンドへの好奇心 |

> ⚠️ 「持ってないの自分だけ説」「今買わないと後悔パターン」は煽りが強すぎてNG

---

## 逆転発想締め型（S-9・ao_kurashi100型）
「隠そうとするより主役にする」「諦めるより活かす」という発想の転換で締める。

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 隠そうとすればするほどゴチャつくから、いっそ主役にするのが正解かも | ごちゃつき・生活感の解決 |
| 諦めてたけど、逆にこれでよかった | 失敗→転換の締め |
| 「どうせ無理」って思ってたやつが、こういうアプローチで解決した | 諦めから希望へ |

> エピソード系投稿の締めに使う。「悩みを諦めず別の角度で解決した」という満足感を伝える。
> ちゃむ。トーンでは「〜のが正解かも」「〜でよかった」と断言しすぎない柔らかさが合う。

---

## 季節タイミング型（S-5型投稿の締めに）
商品の話を「今の季節・今のタイミング」に紐付けて締める。

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 今年の梅雨はこれに活躍してもらいます | 梅雨・室内遊び系 |
| 夏休みの前に揃えといてよかった | 夏休み・長期休暇前 |
| 秋から保育園の子は今が準備タイミング | 入園・進級シーズン |
| 年末年始のおうち時間に大活躍しそう | 冬休み・おうち時間 |

> 商品を「生活の一部として使い続けている」印象を作る。
> 「活躍してもらいます」という表現は主語が商品になり自然な口語になる。

---

## 感情行動報告型（S-2型投稿の締めに）
「私がこう動いた」という自分の行動でURLに誘導する。押しつけ感ゼロ。

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 欲しくて帰って速攻調べたわ | 衝動購入・即決系の共感 |
| 気になって帰り道にポチってた | 衝動感・テンポ感 |
| 見た瞬間これだと思って即注文した | 直感・確信の共有 |
| 帰り道ずっとこれのこと考えてた | 購入欲の滲み出し |
| 翌日には届いてたっていう | スピード感・後日談 |

> 日常シーン報告型フック（型S-2）と組み合わせると自然な流れになる。
> 「私が先に動いた」事実をCTAにすることで読者も追いかけやすくなる。

## ちゃむ。コメント欄でのCTA配置ルール

### 基本構成
```
[コメント欄本文 2〜4行]
[CTA 1行]（PR）
[URL]
```

### CTA選択の優先順位
1. **普段**：カジュアル誘導型・背中押し型・体験談型から選ぶ
2. **セール中**：緊急性訴求型・お得感訴求型を追加
3. **新商品・バズ前**：希少性訴求型
4. **比較検証した投稿の末尾**：比較優位型

### NG組み合わせ
- 緊急性訴求型をセール外で使う → 信頼が落ちる
- FOMO型を多用する → 押しつけ感が出てちゃむ。らしくない
- 「実質タダ」「人生損してる」など大げさすぎる表現 → AI感・広告感が出る

### ちゃむ。でよく使うCTAの実例
- 「7月に入ると毎年品薄になるから早めに準備しておきたいやつ（PR）」← 希少性+体験談の融合
- 「きちんと洗って乾かしてから入れると長く綺麗に残せる（PR）」← 有益情報で締める型
- 「朝に10分冷凍庫で冷やして持たせるだけ、洗えるし普通のリュックにも使える‼」← 使い方+背中押しの融合
