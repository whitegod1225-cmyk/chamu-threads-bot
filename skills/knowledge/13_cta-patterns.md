# 13_cta-patterns.md ── CTA集（コール・トゥ・アクション）

購入者限定特典「CTA集」をちゃむ。用にknowledge化したもの。
アフィリエイト投稿のコメント欄末尾（URLの直前）に使う。

> **使い方**：コメント欄の最後、URLの1行前に置く。
> ちゃむ。トーンに合わせて「友人への語りかけ」レベルに調整する。
> 「実質タダ」「持ってないの自分だけ説」などFOMO煽りが強いものは使わない。

---

## カジュアル誘導型（ちゃむ。で最もよく使う型）
自然に誘導できる。押しつけ感がなくちゃむ。トーンに最も馴染む。

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 気になる人はこちら | 万能・自然な誘導 |
| リンク貼っとくね | 友達感覚の誘導 |
| ここで買えるよ | 購入導線の提示 |
| チェックしてみて | 軽い推奨 |
| 詳細は下のリンクから | シンプルな案内 |

---

## 背中押し型（迷っている人を後押し）
「見るだけでもいい」「ハードルが低い」形で誘導する。

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| とりあえずページだけでも見て | 低いハードル・クリック促進 |
| 見るだけならタダだし | リスクゼロ訴求 |
| 気になったら覗いてみて | 軽い誘導 |
| 迷ってる人は一回見た方がいい | 決断の後押し |

---

## 体験談型（「うちはここで買った」感）
実体験ベースでちゃむ。らしさが出る。信頼感UP。

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 私はここで買いました | 実体験の信頼性 |
| 買ってから毎日使ってる | 日常使いアピール |
| リピート〇回目だけど飽きない | 継続使用の証明 |
| 友達にも勧めまくってる | 口コミの自然な流れ |

---

## 緊急性訴求型（セール・イベント期間限定で使う）
楽天スーパーSALE・マラソン・5と0のつく日など、イベント時に使う。
**普段使いしすぎると信頼が落ちるので、本当にセール中のときだけ使う。**

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 今だけ半額になってる | セール・タイムセール |
| セール終わる前に見て | 期間限定の強調 |
| 在庫残りわずかっぽい | 品薄状況 |
| 明日には値段戻るかも | 価格変動の可能性 |
| このタイミング逃したら後悔する | 機会損失回避 |

---

## お得感訴求型（コスパ・節約スタイルのちゃむ。と相性◎）

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| コスパやばすぎて笑った | コストパフォーマンス強調 |
| この値段でこのクオリティはバグ | 異常な安さの強調 |
| ポイント爆盛りキャンペーン中 | ポイント還元アピール |
| 1000円以下で買えるやつ | プチプラ訴求 |

---

## 希少性訴求型（早めに知ってほしい系）

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| バズる前に教えとく | 先行者利益 |
| 知ってる人だけ得してる | 情報格差の提示 |
| これ見つけた時即買いした | 発見の喜び |
| 売り切れる前にチェック | 品切れリスク |
| 再入荷したから急いで | 再入荷の希少性 |

---

## 比較優位型（比較・検証後の結論として）

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 高いやつと変わらないクオリティ | コスパの証明 |
| 他の商品試したけど結局これ | 比較後の結論 |
| 同じ値段なら断然こっち | 選択の明確化 |
| ランキング上位の理由が分かる | 人気の裏付け |

---

## 問題解決型（悩み解決の締めに）

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| 〇〇で困ってる人に教えたい | ターゲット明確化 |
| もっと早く知りたかった | 後悔の裏返し（共感を引き出す） |
| 同じ悩みの人は見て | 共感からの誘導 |
| これさえあれば大丈夫 | 安心感の提供 |

---

## 数字訴求型（信頼性・実績を示す）

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| レビュー1万件超えの実力 | 社会的証明 |
| ☆4.8の高評価には理由がある | 評価の信頼性 |
| 1年使っても壊れない | 耐久性アピール |
| 〇〇円引きクーポンあり | 具体的割引額 |

---

## FOMO型（乗り遅れ回避）
**ちゃむ。トーンでは基本使わない。使う場合は最も穏やかなものだけ。**

| CTA例文 | 使える場面・効果 |
|--------|----------------|
| みんな買ってるやつ | 流行への乗り遅れ回避 |
| バズってる理由を確認 | トレンドへの好奇心 |

> ⚠️ 「持ってないの自分だけ説」「今買わないと後悔パターン」は煽りが強すぎてNG

## ちゃむ。コメント欄でのCTA配置ルール

### 基本構成
```
[コメント欄本文 2〜4行]
[CTA 1行]（PR）
[URL]
```

### CTA選択の優先順位
1. **普段**：カジュアル誘導型・背中押し型・体験談型から選ぶ
2. **セール中**：緊急性訴求型・お得感訴求型を追加
3. **新商品・バズ前**：希少性訴求型
4. **比較検証した投稿の末尾**：比較優位型

### NG組み合わせ
- 緊急性訴求型をセール外で使う → 信頼が落ちる
- FOMO型を多用する → 押しつけ感が出てちゃむ。らしくない
- 「実質タダ」「人生損してる」など大げさすぎる表現 → AI感・広告感が出る

### ちゃむ。でよく使うCTAの実例
- 「7月に入ると毎年品薄になるから早めに準備しておきたいやつ（PR）」← 希少性+体験談の融合
- 「きちんと洗って乾かしてから入れると長く綺麗に残せる（PR）」← 有益情報で締める型
- 「朝に10分冷凍庫で冷やして持たせるだけ、洗えるし普通のリュックにも使える‼」← 使い方+背中押しの融合
