# 13_cta-patterns.md ── CTA集（投稿の締め・行動誘導）

> **このファイルの使い方**：通常投稿なら「通常投稿CTA」から、アフィリなら「アフィリCTA」から選ぶ。
> CTAは1投稿1種類まで。フォロー・保存・コメントを全部盛りにしない。
> ちゃむ。の口語（タメ口・文末「。」なし）に必ず言い換えてから使う。

229本の実績分析（2026年6月時点）に基づく改訂版。
CTAの原則：**押しつけない・答えやすくする・「私も」と言いたくなる余白を残す**。

# 第1部：通常投稿CTA（メイン・投稿の8割）

## 1. フォロー誘導型

「こういう投稿を毎日しています」型の自然な1行のみ使う。
「フォローしないと損」系の煽りは全面禁止。

| CTA例文 | 使える場面 |
|--------|-----------|
| 保育士16年の声かけ、毎日ひとつずつ置いてくね | 声かけ・知識系の締め |
| 明日も「今日から使える一言」置いとくね | 「代わりに言う言葉」型の締め |
| 育児の罪悪感が軽くなる話、毎日してます | 再定義・救い系の締め |
| 3児の母のリアル、だいたい毎日こんな感じで書いてる | あるある系の締め |
| 続きはまた明日書くね | シリーズ・連載感を出す締め |

**使い方メモ**
- 「フォローしてね」と直接言うより「毎日ここにいる」ことを伝えるほうが効く
- 投稿内容と誘導文を一致させる（声かけ投稿→声かけを毎日、と約束する）
- フォロー誘導は3〜4投稿に1回で十分。毎投稿つけると営業感が出る

## 2. 保存誘導型

「あとで見返す理由」をセットで示す。まとめ・リスト投稿の締めに使う。

| CTA例文 | 使える場面 |
|--------|-----------|
| 癇癪の最中は思い出せないから、保存して落ち着いてから見返して | 声かけリスト系 |
| 次の「早くして！」の前に読み返せるように置いとく | 代わりの言葉系 |
| イヤイヤ期真っ最中の人は、たぶん何回も見返すやつ | 知識まとめ系 |
| スクショでもいいから手元に置いといて | 実用リスト系 |

**使い方メモ**
- 「保存推奨」「保存必須」だけの一言は機械的でNG。「いつ見返すか」を必ず添える
- 保存誘導が効くのは実用系のみ。共感・あるある投稿には付けない

## 3. コメント誘導型（答えやすい質問形式のみ）

「〇〇な時どうしてた？」型だけ使う。考えないと答えられない質問はNG。

| CTA例文 | 使える場面 |
|--------|-----------|
| みんなは「早くして」の代わりになんて言ってる？ | 声かけ系 |
| イヤイヤ期、一番きつかったのって何歳だった？ | 経験シェア系 |
| うちだけ？って思ってるやつ、あったら教えて | あるある系 |
| 今日の夜ごはん、手抜きした人〜 | 共感・挙手系 |
| 上の子の時これ知ってた人いる？ | 知識系 |

**使い方メモ**
- 答えが一言で済む質問にする（何歳・なんて言う・挙手）
- 「どう思いますか？」「意見を聞かせて」は答えるハードルが高くてNG
- コメントが来たら返す前提の投稿にだけ付ける（/replyerの対応パターンを確認）

## 4. 救い締め型（CTAなしの締め）

救い系・再定義系の投稿は、行動誘導せず「許可の一言」で終わるほうが伸びる。
これも立派なCTA（心に残る→プロフィールに来る）として扱う。

| 締めの例文 | 使える場面 |
|-----------|-----------|
| 今日もおつかれさま。それだけで十分だよ | 夜の救い系 |
| 怒っちゃった日も、ちゃんと母やってるから | 罪悪感テーマ |
| 明日はもうちょっとだけ楽になるといいね | しんどい系 |
| 大丈夫、みんな同じとこでつまずいてる | 悩み共感系 |

**使い方メモ**
- 救い系投稿にフォロー誘導や質問を付けると台無しになる。何も求めずに終わる
- 「頑張って」は使わない。「おつかれさま」「十分だよ」の労いで締める

# 第2部：アフィリCTA（投稿の2割）

コメント欄の最後、URLの1行前に置く。

## 5. カジュアル誘導型（最もよく使う）

| CTA例文 | 使える場面 |
|--------|-----------|
| 気になる人はここ見てみて | 万能 |
| リンク貼っとくね | 友達感覚の誘導 |
| うちが買ったのはこれ | 実体験ベース |
| とりあえずページだけでも見てみて | 迷ってる人の背中押し |

## 6. 体験談型

| CTA例文 | 使える場面 |
|--------|-----------|
| 私はここで買って、3人でずっと使ってる | 長期使用の信頼 |
| リピート何回目か数えてないけど飽きない | 高リピート商品 |
| 上の子の時に知りたかったから、置いとくね | タイムラグ後悔系 |

## 7. 有益情報締め型（ちゃむ。の得意技）

売り込みじゃなく「使い方のコツ」で締めてからURLに繋ぐ。

| CTA例文 | 使える場面 |
|--------|-----------|
| きちんと洗って乾かしてから入れると長く綺麗に残せる | お手入れのコツ |
| 朝10分冷凍庫で冷やして持たせるだけでいける | 使い方の具体化 |
| 7月に入ると毎年品薄になるから早めがいいよ | 体験ベースの時期情報 |

## 8. 緊急性訴求型（セール中のみ・普段使い禁止）

楽天スーパーSALE・マラソン・5と0のつく日など、本当にセール中のときだけ。

**セール・イベント型**（oji_to_moji式）

| CTA例文 | 使える場面 |
|--------|-----------|
| マラソン中に買わないと損するやつ | マラソン期間 |
| 5と0のつく日の今日が一番安い | ポイント5倍デー |
| クーポン今日までだったから急いで貼っとく | クーポン期限 |
| 前回これ買い逃して後悔したから今回は早めに言う | 体験ベースの損失 |
| スーパーSALE前に仕込んでおくやつ | SALE直前 |

**季節型**（oji_to_moji式）

| CTA例文 | 使える場面 |
|--------|-----------|
| 去年も夏前には品薄になってたから早めに言う | 夏グッズ・日焼け止め等 |
| 花粉シーズン入ってからじゃ遅いから | 春の花粉グッズ |
| 梅雨入り前に持っておきたいやつ | 梅雨・雨具系 |
| 保育園の入園準備、3月になってからじゃ間に合わない | 入園グッズ |

**希少性型**（oji_to_moji式）

| CTA例文 | 使える場面 |
|--------|-----------|
| 前回は即完売してたから早めに見て | 人気で品切れしやすい商品 |
| 在庫残りわずかって出てたから急いで貼っとく | 品薄状況 |
| 再販がいつかわからないから今のうちに | 復刻・限定商品 |

**「今買う理由」の作り方**
抽象的な「お得です」「セール中」は0点。読者が得する・損する場面を具体的に描く。

| ❌ 抽象 | ✅ 具体 |
|--------|--------|
| セール中だよ | マラソン中に買わないと損するやつ |
| お得です | オムツ代2000円浮く計算だった |
| 安くなってます | 前回買い逃して定価で買った私が言う |
| 季節的にそろそろ | 去年も夏前には売り切れてたから早めに言う |

# CTA配置ルール

## 通常投稿

```
[本文 200〜350文字]
[コメント欄：続き・補足]
[CTA 1行]（フォロー誘導 or 保存誘導 or 質問 or 救い締めのどれか1つ）
```

- 投稿の型でCTAを選ぶ：実用リスト→保存 / あるある→質問 / 救い系→救い締め / 知識系→フォロー誘導
- 同じ種類のCTAを3投稿連続で使わない

## アフィリ投稿

```
[コメント欄本文 2〜4行]
[CTA 1行]（PR）
[URL]
```

- 普段：カジュアル誘導・体験談・有益情報締めから選ぶ
- セール中のみ：緊急性訴求を追加

# NGルール（全CTA共通）

- 「実質タダ」「持ってないの自分だけ」「今買わないと後悔」→ FOMO煽り全面禁止
- 「フォロー必須」「保存しないと損」→ 命令・煽りの誘導は禁止
- 「参考になれば嬉しいです」「いかがでしたか」→ AI定型文は全面禁止
- セール外での緊急性訴求 → 信頼が落ちる
- 1投稿に複数CTA（フォローして＋保存して＋コメントして）→ 全部薄まる
- 「〜ましょう」「〜してくださいね」→ ちゃむ。の口語ではない
