## 型①：「バグ」コスパリアクション型（poni__rooooom）

### 投稿本文

**1枚目**
えっ、バグってる？
YOUNG&OLSENのバッグが付録って？？？

**コメント欄**
この価格で手に入るのバグ……（PR）a.r10.to/hPhyp…

### 解析

**フック構造：**
```
えっ、バグってる？
[ブランド名/商品]が[意外な条件（付録/激安）]って？？？
→ コメント欄：この価格で手に入るのバグ……
```

**技法まとめ：**
- 「バグ」= お得すぎてシステムエラーかと思うレベル、という俗語的表現
- 超短い2行で完結・説明一切なし
- 「？？？」の三連続で驚き・困惑を表現
- コメント欄も最小限（「この価格で手に入るのバグ……」だけ）

**ちゃむ。への転用例：**
```
えっ、バグってる？
あの電動歯ブラシがスーパーセールでこの値段って？？？
---（コメント欄）
歯医者代の方が高くなるのバグ……（PR）
```

**使える場面：** ブランド品の付録・大幅割引・予想外のセール価格

## 型②：URLまとめキュレーター型（juju_kurashi_kodomo）

### 投稿本文

❶ 減煙ホットプレート50%OFF → URL
❷ ランドリーバスケット30%OFF → URL
❸ コンパクトホットプレート50%OFF → URL
❹ ホットサンドメーカー50%OFF → URL
❺ スチームオーブントースター50%OFF → URL
❻ ランドリーバスケット大容量2段30%OFF → URL

### 解析

**フック構造：**
```
[ジャンル名] [割引率]OFF → URL
（繰り返し・説明なし）
```

**技法まとめ：**
- 商品説明ゼロ・割引率とURLだけ
- 「セールのキュレーター」として機能する
- 読者は自分が欲しいカテゴリだけタップすればいい
- セール期間中に複数商品を一気に届けられる
- 投稿制作コストが極めて低い

**構成の原則：**
- 1投稿に3〜6商品（多すぎると見にくい）
- ジャンルを揃えると「このアカウントは〇〇に詳しい」と認識される
- 割引率を先に書く（「50%OFF」が最初に目に入る）

**ちゃむ。への転用例（育児グッズセール期間）：**
```
楽天スーパーセール、育児グッズ厳選↓
❶ 電動歯ブラシ（子ども用）50%OFF → URL
❷ 鼻吸い器セット40%OFF → URL
❸ 防水シート30%OFF → URL
❹ チャイルドシート遮熱カバー50%OFF → URL
（PR）
```

**使える場面：** セール期間中のまとめ投稿・カテゴリ特化まとめ

## 型③：雑談→商品転換型（miniriochan）

### 投稿本文

**1枚目（雑談パート）**
お疲れさまー🫶🏻✨
今日も暑かったねー🥹
5月もう終わりだねー🥹

↓改行で商品情報へ転換：

【5/31価格改定】まだ間に合う！
甘いものが苦手なお父さんへ贈る、長崎限定の『最高峰コーヒーゼリー』がガチでお洒落✨

### 解析

**フック構造：**
```
[日常の一言・季節の挨拶]（2〜3行）
↓（転換）
[値上げ・セール情報]まだ間に合う！
[ターゲットを絞った商品説明]
```

**技法まとめ：**
- 日常の挨拶・雑談でフォロワーとの温度感を作ってから商品情報へ
- 「今日も暑かったね」→「だから夏グッズを」のような自然なつながりが理想
- 強引な転換は「急に宣伝か」となるのでスムーズな文脈が必要
- miniriochanはここからの文体がAI感強い（NG）→ちゃむ。は口語で続ける

**ちゃむ。の転用時の注意：**
- 雑談→転換の「つながり」を作る（「暑くなってきたから」「月末だから」等）
- 転換後もAI感を出さずちゃむ。文体を維持する

**良い転換例：**
```
今日もおつかれ〜🙌
もう6月だね、ほんとあっという間

そういえば今夜からスーパーセール。
去年の夏に買ってよかったやつ置いとくね↓（PR）
```

**使える場面：** セール直前の夜・月末・季節の変わり目

## 補足：非アフィ共感投稿（miniriochan 名言）

アフィリエイト投稿の間に挟む「関係性維持投稿」の例。

```
人生って
頑張ることより
「無理しないこと」
の方が
大事だったりする。
```

**なぜ重要か：**
- アフィ投稿だけ続けると「売りたい人」のアカウントになる
- 共感・応援投稿を挟むことで「この人好き」という感情を維持する
- ⑩フォローしたくなる心理14選（knowledge参照）の「共感投稿：同一視（自分の気持ちを代弁してる）」に対応

**ちゃむ。の通常投稿（型2・型11・型7等）がこの役割を担う**
→ アフィ投稿5本に1本のペースで通常投稿を挟む設計は正しい
