## 型の解析

### フック構造
```
ガチで[対象]を[成果]するなら
❶[時期・月齢]なら「[具体的アイテム名]」
❷[時期・月齢]なら「[具体的アイテム名]」
...（繰り返し）
は必須。まだまだあります。[フォローCTA]
```

### 技法まとめ

**①「ガチで〇〇するなら」冒頭断言**
- 「ガチで」が本気度と信頼感を一言で表す
- 続きを読まないと損と思わせる

**②月齢・時期別の具体性**
- 「〇〇ヶ月なら〇〇」という具体性が保存率を上げる
- 「自分の子の月齢のところだけ見る」という使われ方
- 保育士の専門性と相性が良い

**③「まだまだあります」で続きへの期待感**
- リストが途中で終わる示唆
- フォローすれば続きが見られると思わせる

**④フォロー誘導で締める**
- 「本気で〇〇したいママは必ずフォローしてくださいな」
- 「本気で」というターゲット絞りでフォロワーの質を上げる

### ちゃむ。への転用例

**月齢別まとめ（通常投稿として使う）：**
```
ガチで0歳の発達を伸ばすおもちゃなら
❶0〜3ヶ月なら「〇〇」
❷3〜6ヶ月なら「〇〇」
...
は外せない。まだまだあります。本気で発達を伸ばしたいママはフォローしといて。
```

**アフィリエイトとの連携：**
- 有益まとめ投稿でフォロワーを増やす
- 翌日の投稿でそのリストの商品のアフィリエイト投稿を出す（「昨日の絵本リスト、楽天で買えるやつまとめた↓」）

### 注意点
- 単体ではアフィリエイトURLなし（有益投稿として使う）
- 翌日のアフィ投稿への橋渡し役として設計する
- 「元保育士が選ぶ」という冒頭にすると権威性がさらに高まる
