## 型①の解析：生活日記型

### 全体構造
```
[今日の育児・子育ての様子を日記として書く]
→ 自然な流れで商品名が1回だけ登場
→ コメント欄で商品の特徴を1〜2行＋URL
```

### 技法まとめ

**①「アフィ投稿」に見えない日記として書く**
- 1枚目は完全に「今日の子どもの様子」
- 商品は「〇〇みたいな〇〇ワークは」と自然に1回だけ触れる
- 宣伝感が一切なく「普通に使ってる」感だけが伝わる

**②親の試行錯誤・葛藤を入れる**
- 「ぐ〜っと堪えてます、私が😂」
- 「完璧な親」ではなく「一緒に悩んでる親」のポジション
- 読者が「わかる！」となり親近感が最大になる

**③コメント欄は最小限**
- 商品の特徴を1〜2行 + URL だけ
- 長い商品説明は不要。すでに1枚目で「使ってる様子」が伝わってるから

**ちゃむ。への転用（最も相性が良い型）**
ちゃむ。はすでに育児日記スタイルの投稿を書いている → そこに商品を自然に登場させるだけ。

```
小2の娘、今日の仕上げ磨きでついに「お母さん磨いて」って言ってきた
1ヶ月前まで逃げ回ってたのに😂
電動歯ブラシに変えてから「自分で押してみる」ってなって、なんか習慣になったみたい

コメント欄：
振動が面白いんだろうな〜と思って。毎晩の格闘がなくなって正直助かってる🙌（PR）a.r10.to/h5hiAq
```

**注意点：**
- 1枚目に商品名を入れる場合は「さりげなく1回」に留める
- 商品を褒める言葉は1枚目に入れない
- 「今日こんなことがあった」という実体験として書く

## 型②の解析：「ここ一択」条件断言型

### フック構造
```
[具体的な条件・状況]のファミリーはここ一択！！！
[お得な理由①] + さらに [セール情報]
→ コメント欄：具体的な割引率・プラン詳細
```

### 技法まとめ

**①「〇〇するなら一択」で迷いをなくす**
- 「ディズニー泊りで行くファミリー」という条件を設定
- その条件の人には「ここ以外ない」と断言することで選択の迷いを消す
- ターゲットが明確なほど「自分に言ってる！」と刺さる

**②「そもそも〇〇なのに + さらに」の二重お得**
- 「元々お財布に優しい」（固有価値）＋「セールで更に安い」（今だけ）
- 二段階のお得感で購買動機を積み上げる

**ちゃむ。への転用例：**
```
子どもと夏の水遊び、公園プールに行くファミリーはこれ一択！！
日焼け止め塗らなくてもいいラッシュガード、ドンキより楽天セールの今が安い
```

```
歯磨きで毎晩戦争してる1〜2歳のいるご家庭はこれ一択！！
電動歯ブラシで「やって〜」に変わった実感を持ってる元保育士として言う（PR）
```
