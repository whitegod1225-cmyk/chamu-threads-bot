## 型の解析

### フック構造
```
[具体的な行動をしようとしている] [ターゲット]ちょっと待って✋
[より良い選択肢の提示]
[具体的な割引率・数字]とかありえない！
```

### 技法まとめ

**①「ちょっと待って✋」行動介入フック**
- 読者が今まさにやろうとしている行動を止める
- ✋の絵文字が視覚的にも「止まれ」のサインになる
- 「あ、私のこと言ってる」と思わせる精密なターゲティング

**②「行動+場所」まで具体化したターゲット描写**
- 「西松屋やベビザラスへ向かおうとしてる妊婦さん」
- 「出産準備中の妊婦」より「今まさに実店舗に行こうとしてる妊婦」の方が刺さる
- 読者の行動フローを先読みして介入する

**③「ありえないわ！ってなってるのぉぉぉ！」の感情リアクション**
- 自分がリアルタイムで驚いている演出
- 「情報を伝えてる人」ではなく「一緒に驚いてる友人」のポジション

**④コメント欄で用途を具体化**
- 「夜中の救世主になる」→使用場面の描写でベネフィットを伝える
- 「売り切れるほど人気の」→希少性で背中を押す

### ちゃむ。への転用例

行動フローを先読みして介入するパターン：

```
「来月から保育園の歯科健診がある」ってメモしながら歯ブラシ選びに迷ってるパパママ、ちょっと待って✋
今の楽天セールで子ども用電動歯ブラシが半額になってる
手磨きで嫌がる子にこそ先に試してほしいやつ（PR）
```

```
そろそろ夏のチャイルドシート対策しようと思って検索し始めてる人ちょっと待って✋
遮熱シート、梅雨明け前に買わないと毎年売り切れるから今がチャンスだよ（PR）
```

### このフックが使える場面
- セール期間中の実店舗 vs 楽天の比較
- 季節の変わり目（夏前・冬前）の準備グッズ
- 行事前（入園・入学・出産準備）の購買タイミング
- 「もうすぐ〇〇する人」が明確に想定できる商品

### 注意点
- ターゲットの行動が具体的・リアルであるほど刺さる
- 「〇〇しようとしてる人」が漠然としているとフックが弱くなる
- ✋は文字で「ちょっと待って」と書く場合でも使えるが、絵文字なしでも成立する
