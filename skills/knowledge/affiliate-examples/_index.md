# affiliate-examples ── アフィリエイト投稿サンプル集

## このフォルダの用途

実際に投稿されたアフィリエイト投稿を保存し、**/affiliate-writer が投稿を生成する際の文体・構成・トーンの型として参照する**。

- `affiliate-research/`（戦略記事）とは別物
- 「こういう書き方で生成してほしい」という見本をストックする場所
- /affiliate-writer はこのフォルダのサンプルを読んで、文体・構成・温度感を模倣する

## 参照タイミング

/affiliate-writer が投稿生成する際に、このフォルダのサンプルを読んで以下を参考にする：
- 1枚目（本文）の冒頭フックの書き方
- コメント欄のつなぎ方・ベネフィットの見せ方
- 絵文字の使い方・密度
- 「（PR）」の入れ方
- 全体の温度感・語尾のトーン

## ファイル命名規則

```
[連番]_[ジャンル一言].md
例：01_日用品.md / 02_ベビーグッズ.md / 03_食品.md
```

## 各ファイルのフォーマット

```
# [タイトル]

**出典**：@アカウント名（Threads / Instagram など）
**保存日**：YYYY-MM-DD
**ジャンル**：[商品ジャンル]
**参考にしたい点**：[冒頭フック / コメント欄の構成 / 絵文字の使い方 など]

---

## 投稿本文

[1枚目の本文をそのまま貼る]

## コメント欄（セルフリプライ）

[コメント欄をそのまま貼る]

---

## 使い方メモ

[どの部分を型として使うか・何が優れているかを一言メモ]
```
