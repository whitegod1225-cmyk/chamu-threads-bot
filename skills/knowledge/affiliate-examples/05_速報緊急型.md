## 型の解析

### フック構造①（値上げ前型）
```
うそでしょ？！[ブランド名]が[日付]から値上げ？！
ここなら[価格感]で売ってる！！！しかも…
```

### フック構造②（セール速報型）
```
え、[ブランド/商品]が[理由]で[割引率]OFFはやばすぎる…
```

### 技法まとめ

**①驚きリアクション開幕**
- 「うそでしょ？！」「え、〇〇はやばすぎる…」
- 読者が「何？！」と思って止まる
- ニュース感・速報感で緊急度が上がる

**②値上げ前の「今のうちかも」**
- 「この値段で買えるのは今のうちかも…」
- 損失回避心理に訴える（今買わないと損）
- 「かも…」で押しつけず読者に判断を委ねる

**③「しかも…」で次スライドへ引っ張る**
- 「しかも…」で終わらせてコメント欄・次スライドを読ませる
- ちゃむ。の「だって」と同じ役割

**④価格感の明示**
- 「1000円台で売ってる」でお得感を一言で表す

### ちゃむ。への転用例
- 「うそでしょ？！〇〇が来月から値上がりするって？！今ここなら〜」
- 「え、楽天スーパーセールで〇〇が〇〇円になってるのやばすぎる…」
- 育児グッズの値上げ・生産終了・セール速報に使える
