## 型の解析

### フック構造（1枚目）
```
[認知度の高い商品]はもうやめた。。。
それよりも、こっちの方が断然良い！！
```

- 「やめた」で読者の興味を引く（何で？と思わせる）
- 有名商品をネガとして使うことで「自分も同じことで悩んでた」と共感させる
- 「。。。」で間を作り、次を読ませる

### コメント欄の構造
```
[認知商品の弱点を問いかけ形式で指摘]（〜じゃない？）
だから、[代替品] がめちゃくちゃ良かった！
[ベネフィット（長時間持続）] + [継続使用実績（去年から）]
[自分も使う] ← 複数用途・信頼性アップ
```

### 特筆すべき技法

**①「〜じゃない？」問いかけ構文**
- 「秒で溶けない？」と読者に問いかける
- 読者が「わかる！」と感じると返信・保存率が上がる
- 相手の体験を代弁しながら自分の主張を進める

**②認知商品のネガ活用**
- アイスリングは知名度が高いため「あの商品より良い」が刺さる
- 比較の文脈を作ることで差別化が明確になる
- ただし悪口にはしない・「見た目はスマートだけど」と認めてから弱点を出す

**③「去年から」継続使用の信頼構築**
- 「去年から使ってる」は「1シーズン使って良かったから今年も」という意味
- 単なる紹介ではなくリピートの実績で信頼感を出す

**④「私も使う」複数用途**
- 「子どもだけでなく自分（犬の散歩）でも使う」
- 用途の広さ＋自分事化でコスパ感が増す

### 使い方メモ
- 「アイスリングはやめた」→「保冷枕はやめた」「市販の離乳食はやめた」等に転用できる
- 「〜じゃない？」は読者巻き込み型の問いかけとして型3・型8にも応用可
- 継続使用の実績提示（「去年から」「3年使ってる」）は信頼構築に効く
