## 型の解析

### 全体構造
```
正直に言う、[セール名]の勝負は今夜
[希少性：年〇回しかない] → 事前準備を強調
[有益なやること番号リスト：①②③]
→ ③でコメント欄に引っ張る
→ コメント欄で詳細 + 「数万円変わる」数値 + URL
```

### 技法まとめ

**①「正直に言う」断言オープニング**
- 本音感・本気度が伝わる
- 「教えてあげる立場」としての権威を感じさせる

**②「年4回しかない」希少性の数値化**
- 「レアなイベント」を数字で可視化
- 「今を逃すと損」という感覚を作る

**③「数万円変わる」実益の数値化**
- 「知ってるか知らないかで数万円差がつく」
- 損失回避（行動しないと損）を具体的な金額で表現

**④リストを途中で切ってコメント欄へ誘導**
- 「①②③」で①②のみ1枚目に→③はコメント欄へ
- 「続き読まなきゃ」と思わせる

**⑤セール会場URLをアフィとして差し込む**
- 商品ではなくセール会場そのものへ誘導
- 会場内での購買が成果になる仕組み

### ちゃむ。への転用例
楽天スーパーセール・お買い物マラソン前に投稿できる有益コンテンツとして使う。

```
正直に言う、今夜から始まるお買い物マラソンの攻略を知ってる人と知らない人で差がつく
❶エントリーをまず必ずする
❷育児グッズは5と0のつく日より今夜の方が安いものがある
❸「ポイント10倍」の店を先に見つけてお気に入りに入れておく
知っておくだけで数千円変わるよ↓（PR）
```

### 活用タイミング
- 楽天スーパーセール前日〜初日
- お買い物マラソン開始前日
- 5と0のつく日の前日夜
