# 超短つぶやき・購買宣言型

**出典**：@lala__closet_（Threads）
**保存日**：2026-06-01
**ジャンル**：葉酸サプリ / マタニティパジャマ
**参考にしたい点**：超短い本音感・購買宣言で読者を引き込む・「セルフツッコミ」で親近感

---

## 投稿例①（葉酸サプリ）

**1枚目**
葉酸サプリは30週までって言われた。
知らなかった、、、
いつもここで半額で買ってるよ

**コメント欄**
上の子の、時もずっとこれ（PR）

## 投稿例②（マタニティパジャマ）

**1枚目**
出産予定日9月だけど明日半額なるから買う。
さすがに毛玉パジャマでは入院出来んw

**コメント欄**
前開きで授乳出来るのもいいよね！↓コレだよ↓（PR）

## 型の解析

### フック構造①（知らなかった型）
```
[知識・情報] + 知らなかった、、、
いつもここで[お得情報]
```

### フック構造②（購買宣言型）
```
[自分の状況・購買理由]
[自虐・セルフツッコミ]（w / 笑）
```

### 技法まとめ

**①「知らなかった、、、」超短つぶやき**
- 3行以内・情報量を最小限に絞る
- 「知らなかった」という本音が読者に刺さる
- 「自分も知らないかも」と思わせてリンクを踏ませる

**②購買宣言で読者を動かす**
- 「明日半額なるから買う」という自分の行動宣言
- 読者が「自分も買おうかな」と思うきっかけになる
- 押しつけではなく「私が買います」というスタンス

**③セルフツッコミで親近感**
- 「さすがに毛玉パジャマでは入院出来んw」
- 自分の状況を笑いにすることで読者との距離が縮まる
- 育児系でよく使われる「あるある」共感ワード

**④コメント欄は超短く**
- 「前開きで授乳出来るのもいいよね！↓コレだよ↓」
- 最低限の機能説明＋URLで完結
- 「長々と説明しない」のも一つの型

### ちゃむ。への転用例

**知らなかった型：**
「子どもの歯ブラシは3ヶ月で変えた方がいいって、保育士1年目で知った。知らなかった親御さんも多かったな、、、うちはいつもここで買ってるよ」

**購買宣言型：**
「来月から値上がりするって聞いて、今日買った。さすがに後で後悔したくないので（PR）」

### 注意点
- 超短い文体は「意味が薄い」にならないように
- 「知らなかった」は読者も知らない情報である必要がある
- コメント欄で情報を補完する設計にする
