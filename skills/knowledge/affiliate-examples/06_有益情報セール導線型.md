## 型の解析

### 全体構造
```
[専門知識・脳科学] でフック
→ 共感ユーモア（「親は飽きる。かなり飽きる（笑）」）
→ [知識の詳細説明] で信頼を積む
→ [「ちょうどいまセール中！」] でアフィ誘導
→ 緊急性（売り切れる前に）
```

### 技法まとめ

**①知識先行で信頼を作ってからセール誘導**
- 1枚目・2枚目前半で有益情報を届ける
- 読者が「この人は知識がある」と思った後にセール情報
- 「売りたい人」ではなく「教えてくれる人」に見える

**②「親は飽きる。かなり飽きる（笑）」共感ユーモア**
- 専門知識の間に親の本音（飽きる）を挟む
- 堅くなりすぎず、距離感が縮まる
- ちゃむ。の文体に転用しやすい

**③「ちょうどいま〇〇！」タイミング合わせ**
- 有益情報とセールのタイミングを重ねる
- 「偶然セールと重なった」感が押しつけを消す
- 実際にセール期間に合わせて投稿するのが最大効果

**④緊急性の締め**
- 「売り切れる前に！」で背中を押す

### ちゃむ。への転用例
ちゃむ。は元保育士の知識がある → 知識型投稿の後半にセール情報を差し込める。

例：
「歯科健診で引っかかる子に共通してたこと（保育現場から5つ）
...（知識）...
ちょうど今、楽天スーパーセールで電動歯ブラシが半額になってる！使ってよかったやつ↓（PR）」

### 注意点
- 知識部分とセール部分のつながりが自然である必要がある
- 知識が薄いまま誘導すると「結局アフィだった」になる
- 知識が7割・アフィ誘導3割のバランスを保つ
