## 型の解析

### 全体構造
```
まだ9割の人が知らない + [認知商品] をやめて [代替品] に変えたら世界変わる
→ ベネフィット列挙
→ 権威の裏付け（皮膚科医の言葉）
→ 第三者証言（ママ友に感謝された）
→ コメント欄：情緒的動機（日本企業応援）+ 追加ベネフィット
```

### 技法まとめ

**①「まだ9割の人が知らない」希少知識フック**
- 「知っている側」への優越感と「知らない側」への焦りを同時に作る
- 続きを読まないと損という感覚

**②権威の三重構造**
| 権威の種類 | 具体的表現 |
|-----------|-----------|
| 自分の体験 | 「世界変わる」「着心地が違う」 |
| 専門家の裏付け | 「皮膚科医もエアリズムは〇〇と言う」 |
| 第三者の反応 | 「ママ友に教えたら2026年で1番いい情報と感謝された」 |

三種類の権威を一つの投稿に重ねることで信頼の密度が上がる。

**③「応援購買」の情緒的動機**
- 「GUNZEは数少ない日本企業だから応援してる」
- 品質だけでなく「買う理由」に情緒を加える
- 読者の「社会に良いことをしたい」気持ちに響く

**④子どもへの言及で育児アカウントと紐付け**
- 「肌弱い子どもも大絶賛」
- 自分だけでなく子どもにも使えることを示す

### ちゃむ。への転用例

保育士経験は「専門家の裏付け」として使える。

```
まだ知らない人が多いんだけど、市販の歯磨き粉って子どもに向かないものがあって。
フッ素濃度が成人用のまま使ってる家庭を保育現場でも見てきた。
先輩保育士から「子ども用は成分が全然違う」と教えてもらってから、うちはずっとこれ。
ママ友に話したら「知らなかった」って驚かれた。↓（PR）
```

### 注意点
- 「皮膚科医も言う」「医師が推奨」等の医療権威は根拠が曖昧だとNG
- ちゃむ。の場合は「元保育士として現場で見てきた経験」が最も使いやすい権威
- 第三者証言は「感謝された」「驚かれた」程度に留める（過剰な賞賛はAI感）
