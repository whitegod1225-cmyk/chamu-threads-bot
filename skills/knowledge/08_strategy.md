# 08_strategy.md ── note連携・バズ構成・KPI戦略

## noteとの連携戦略
- Threadsは「共感・信頼構築」の場。noteは「深掘り・マネタイズ」の場
- 週4投稿程度でnote誘導を自然に組み込む

## 誘導パターン
- 問題提起型：Threadsで問題提起→「詳しくはnoteで」
- リスト予告型：一部だけ投稿→「残りはnoteで」
- 体験談完結＋深掘り誘導型

## バズ投稿の型

### 型1：他人の家で解決型
- 感情：驚き・発見
- 長さ：長め
- 拡散：保存・情報拡散
- 頻度：週1
- 構造：諦めかけていた問題→他者の場で偶然解決→「なんで？」→失敗vs成功の比較表→再現性の証明
- ちゃむ。軸：「保育士なのに我が子には通じなかった」自己矛盾を必ず入れる

### 型2：敗因報告型
- 感情：共感・笑い
- 長さ：超短文
- 拡散：コメント・RT
- 頻度：週2〜3
- 構造：対象読者への呼びかけ→状況一言提示→理不尽な敗因の開示→ゲーム用語で締め
- テンプレ：「[テーマ]のみんなへ／今日の[場面]です／敗因は、[些細な原因]です／対戦よろしくお願いします」

### 型3：現場ハック集型
- 感情：信頼・即実践
- 長さ：中〜長め
- 拡散：保存・参加
- 頻度：週1
- 構造：タイトルでテーマ明示→具体的すぎるハック7〜10個→「保育園で毎日使ってた」出典→参加型CTA
- ちゃむ。軸：「ママに教えてもらった」ではなく「保育園で毎日使ってきた」に変換
- **禁止：各項目の後ろに（）で補足を入れない**。「① 〇〇（△△だから）」のカッコ補足はAI感が出る。補足を書く場合は改行して次の行に置く

### 型4：NG→OK変換型
- 感情：罪悪感→納得・安堵
- 長さ：中め
- 拡散：保存・シェア
- 頻度：週1〜2
- 構造：**NGシーンを1つだけ選ぶ**→具体的な場面・セリフの描写→なぜその言葉が刺さるかを深掘り（保育士視点）→代わりに何をすればよいかを1つ示す→締めの一言
- 注意：NGを複数列挙する箇条書き形式は離脱を招くためNG。1つに絞って深く掘り下げること
- 例：「うちの子、迷惑かけてますよね」という言葉を1投稿で深掘りする
- ちゃむ。軸：保育士16年の権威で「なぜNGか」の説明に圧倒的説得力が乗る。シーンの描写が具体的なほど読者の「あ、これわたしだ」が生まれる

### 型5：神対応どんでん返し型
- 感情：笑い・安堵
- 長さ：中め
- 拡散：コメント・シェア
- 頻度：週1
- 構造：自分の失敗提示→他者の神対応シーン→期待が最高潮→どんでん返し（神対応の人も実は〇〇だった）→安堵の締め一言
- ちゃむ。軸：「先輩保育士も最初の5年は同じだった」という自己開示

### 型6：頑張ったのに空振り型
- 感情：複雑な本音・連帯
- 長さ：中め
- 拡散：コメント・共感RT
- 頻度：週2
- 構造：自分が限界だった状況→全部空振りした事実→第三者があっさり解決→複雑な心の声→無自覚な追い打ち→「全国の母たち、わかるよね」で締め
- ちゃむ。軸：「保育士として1000人見てきた私が、我が子には通じなかった」

### 型7：概念の誤用型
- 感情：笑い・共感
- 長さ：超短文
- 拡散：RT・コメント爆発
- 頻度：週1
- 構造：子ども専用の概念や育児用語を大人・自分自身に転用→説明しない
- ちゃむ。軸：保育士用語を自分に診断として使う。「プロが自分に診断を下す」構造

### 型8：保育士権威×NG/OK型
- 感情：信頼・実用
- 長さ：中〜長め
- 拡散：保存・シェア
- 頻度：週1
- 構造：経験年数＋具体的現場文脈→差は対応だけという断言→NGコース箇条書き→OKコース箇条書き→プロの本音＋逆張り結論
- 注意：競合が多い型。必ず自分の失敗エピソードを混ぜること。体験が乗った瞬間にちゃむ。だけの投稿になる

### 型9：助けて募集型
- 感情：弱さ・連帯
- 長さ：超短文
- 拡散：コメント爆発
- 頻度：週1
- 構造：全員が持っている体験への質問→投稿者自身の今の本音（弱さの開示）
- ちゃむ。軸：保育士が「心折れそう」と言う破壊力は通常の3倍

### 型10：主語ずらし報告型
- 感情：脱力・共感
- 長さ：超短文
- 拡散：RT・コメント
- 頻度：週1
- 構造：やろうとしていた具体的行動→主語をずらした擬人化（自分のせいにしない）→時間の経過を客観的に報告
- テンプレ：「〇〇しようとしていた[気力/意欲/共感力]が／[擬人化表現]して／早〇時間が経過……」

### 型11：謎の場面目撃型
- 感情：笑い・愛しさ
- 長さ：長め
- 拡散：コメント・シェア
- 頻度：週1
- 構造：意味不明な場面の目撃→「え、どういう状況？」と読者の疑問を代弁→謎の背景を順番に説明→伏線回収（実は〇〇だった）→笑いと愛しさが同時に来る感情着地→自分の反応の実況で読者を巻き込む
- ちゃむ。軸：保育園で目撃した謎の場面、または自分の子育ての珍場面

### 型12：フォロー価値提示型（月1回）
- 感情：安心・期待
- 長さ：中め
- 拡散：フォロー転換
- 頻度：月1回（毎月第1土曜が目安）
- 構造：「ちゃむ。の投稿をまとめておく」という切り口→カテゴリ別に何を届けているかを箇条書き→「必要な時に読めるようにフォローして」で締める
- 注意：コメント欄なし。フォロー転換だけに集中させる。宣伝臭が出たら即書き直す。

### 型A：自己紹介型（フォロー転換専用）
- 感情：共感・親近感・安心
- 長さ：5〜7行（短めが鉄則）
- 拡散：フォロー転換
- 頻度：**10投稿ごとに1本**（post-queue.mdへの追記時に自動チェック）
- カテゴリ：A（フォロー転換）
- **目的**: 新規流入ユーザーに「この人どんな人？」を一瞬で伝えてフォローさせる
- **絶対NG**：商品名・アフィリエイトリンク・「稼ぎたい」などの本音・完璧すぎる自己アピール

#### サブタイプA-1：まとめ自己紹介型
- 構造：弱み・本音 → 価値観 → 発信ジャンル → ターゲット → フォロー誘導（ゆるく）
- コメント欄：過去に反応が良かった投稿のスクリーンショット3〜5枚（リンクなし・商品名なし）
- 効果：人柄とジャンルが一目で伝わる。コメント欄の画像がアカウントの雰囲気を補完する

```
テンプレート（ちゃむ。版）：
保育士16年やってたのに、
自分の子には全然通じなくて😂

ここでは、
現場で学んだ「育児の知識」より
「気持ちが楽になる育児の話」を届けてます。

プロでも毎日悩む。
一緒に考えながら育ててくれる人とつながりたい。
気が向いたらフォローしてください。
```

#### サブタイプA-2：あるある日常自己紹介型（やわらか型）
- 構造：あるある1行目 → 属性を自然に → 好きなもの・こだわり（固有名詞で） → 理想・世界観 → ゆるい誘導
- コメント欄：なし（文章だけで人柄を伝える）
- 効果：営業感ゼロ・親近感が最大になる。最初の1行であるあるに入るのがポイント

```
テンプレート（ちゃむ。版）：
「こうすればいい」って頭では分かってても、
自分の子の前ではぜんぶ吹っ飛ぶ🥲

元保育士で3人の母。
プロとしての知識と、等身大の失敗談を正直に発信してます。

頑張りすぎず、楽しめる育児を目指して。
同じように悩んでるママ、仲良くしてください。
```

#### 型Aに書くちゃむ。の要素
| 要素 | ちゃむ。の内容 |
|------|------------|
| 弱み・本音 | 「保育士なのに自分の子には通じなかった」「感情的になることもある」 |
| 価値観 | 「がんばりすぎない育児」「褒めて伸ばす」「知識より共感」 |
| 属性 | 元保育士16年・3児の母・30代 |
| 好きなもの | 育児グッズ探し・子どもとの時間・節約育児 |
| 発信ジャンル | 育児知識・育児グッズ・保育士目線の子育てアドバイス |
| ターゲット | 「育児で正解を求めて疲れているママ」「ワンオペで消耗してるママ」 |
| フォロー誘導 | 「気が向いたらフォローしてください」「仲良くしてください」 |

### 型13：楽天アフィリエイト体験談型
- 感情：問題解決・信頼・共感
- 長さ：**1枚目3行以内**（超短文）＋ コメント欄で展開
- 拡散：保存・コメント
- 頻度：週1〜2回（全投稿の15%以内。連続して入れない）
- カテゴリ：E（節約・グッズ・生活系）
- **キュー**：通常投稿（post-queue.md）とは別の `affiliate-queue.md` で管理する
- 構造：
  - **1枚目（本文）3行以内**：場面描写・共感・フックのみ。商品名・スペック・価格は書かない。最終行は「だって」「それが」など中途半端に切り、コメント欄を自然に読みたくなる仕掛けにする
  - **コメント欄（セルフリプライ）**：1枚目の続きを冒頭で受けてから、具体的なベネフィット／使用感 ＋「（PR）」明記 ＋ 楽天アフィリエイトURL
- 注意点：
  - 「これを買えば解決！」はNG。「うちはこれにした」くらいの温度感にする
  - 「おすすめです！」「ぜひ！」連発はNG（05_affiliate.md参照）
  - 商品名より「どんな問題が解決されたか」をコメント欄で伝える
  - 保育士として現場で似たグッズを見てきた経験があれば一言添える
  - コメント欄に「PR」明記＋楽天リンクを置く（本文には絶対入れない）
  - 本文に価格・スペックを書かない（広告臭が出る）

### 型13B：楽天アフィリエイトまとめ投稿型（ツリー型）
- 感情：参考・保存・信頼
- 長さ：本文3行以内（問題提起）＋ コメント欄で商品3〜5個列挙
- 拡散：保存・コメント
- 頻度：週1回（型13と合わせて週2本以内。連続しない）
- カテゴリ：E（節約・グッズ・生活系）
- **キュー**：affiliate-queue.md で別管理
- 構造：
  - **1枚目（本文）3行以内**：「〇〇に悩むママへ」など対象を明確にした問題提起のみ。最終行は「全部ここにある」「リストにした」など中途半端に終わらせる
  - **コメント欄（セルフリプライ）**：テーマ名 → 商品を❶❷❸形式で3〜5個 → 各商品1行説明 → 楽天アフィリエイトURL（商品ごと）→（PR）明記
- 注意：各商品説明は1行以内。育児ジャンルは「見た目」より「言葉」で売るため、悩みへの共感から入ることが必須。画像よりも文章の構成力が問われる型

## 型の運用サイクル（1週間×5投稿）
- 月：型3か4（保存系・週の始めに有益情報）
- 火：型2か7（超短文・気軽に読める）
- 水：型1か5（中長文・読み応え）
- 木：型6か8（共感系・週の疲れに刺さる）
- 金：型9（募集型・週末に向けてコメント盛り上げる）
- 土（月1回）：型12（フォロー価値提示型）

超短文型（2・7・9・10）を週の隙間に差し込むと投稿が止まらなくなる。

## ちゃむ。最強の差別化軸
「元保育士として知ってた。でも自分の子どもには全部通じなかった。」
この自己矛盾はちゃむ。だけが持てる武器。全型で使える。

- 3ヶ月：2,000フォロワー
- 6ヶ月：5,000フォロワー
