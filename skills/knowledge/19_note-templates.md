# 19_note-templates.md ── note記事・誘導投稿テンプレート

> **このファイルの使い方**：noteを新しく書くとき・Threadsからnoteへ誘導するときに参照する。
> [テーマ][価格][URL]の変数部分を差し替えて使う。

Fable 5生成・2026-07-04

---

## 全体の設計思想（mojibake_writing式）

```
無料note（入口）→ Threads定期シェア → 反応が良いnoteを有料にアップグレード → 月1訴求
```

- 有益情報4：経験談5：宣伝1のバランスを守る
- noteはThreadsで「売る」のではなく「置いておく」温度感で出す
- URLは本文には入れない。必ずコメント欄の末尾に置く

# 第1部：Threads→note誘導投稿テンプレート

## パターンA：無料note誘導型
> 本文で共感→コメント欄でnoteの存在をさりげなく伝える

**本文（約240文字）**
```
[テーマ]のこと、検索すればするほど
わからなくなるのって私だけじゃないよね

正解っぽい情報はいっぱい出てくるのに
うちの子には当てはまらない気がして
夜中にスマホ握ったまま固まってた時期があった

保育士やってた私でもそうだったから
今まさに悩んでるママが
自分を責める必要なんてほんとにないよ

同じところでつまずいてる人
コメント欄も見てみて
```

**コメント欄**
```
[テーマ]で悩んでたころの私に渡したくて
これまで聞かれたこと・答えてきたことを
noteに1個にまとめて置いといたよ

無料だから、夜の授乳中とかに
ゆっくり読んでもらえたら嬉しい

[無料noteURL]
```

## パターンB：有料note誘導型
> 本文で保育士としての本質を1行だけ出す→コメント欄で内容紹介

**本文（約220文字）**
```
16年保育士やって気づいたのは
[テーマ]で苦しんでるママほど
ちゃんと子どもを見てるってこと

手を抜いてる人は悩まないんだよ
悩むのは、真剣だから

私も3人育てながら
「保育士の知識が自分の子に通じない」って
何回も泣いたからわかる

その悩み方、間違ってないよ
```

**コメント欄**
```
[テーマ]について、園で見てきたことと
自分の子育てでぶつかったことを
全部noteに書いた

正直、Threadsには書ききれない
失敗談までそのまま載せてる

「頑張ってるのにうまくいかない」って
思ってる人に届けたくて書いたやつです

[価格]円 → [有料noteURL]
```

## パターンC：シリーズ締め誘導型
> 3〜5本シリーズの最終本コメント欄末尾に追加する

**コメント欄末尾に付け足す（3〜4行）**
```
このシリーズ、ここでいったんおしまい

でも正直、5本じゃ書ききれなかった部分が
けっこうあって、そこを全部noteにまとめた

シリーズ読んでくれた人なら
たぶん一番知りたかったところだと思う

[noteURL]
```

## 誘導投稿の使い分けルール

| パターン | 使うタイミング | 頻度 |
|---------|-------------|------|
| A（無料note） | 通常投稿のコメント欄から自然な流れで | 週1〜2本 |
| B（有料note） | 反応が良かった投稿の翌日or翌々日 | 月2〜4本 |
| C（シリーズ締め） | 3本以上のシリーズ最終本 | シリーズごとに1回 |

- 同じパターンを3投稿連続で使わない
- パターンBは通常投稿10本に1本以下

# 第2部：無料note骨格テンプレート

## 構成（目次）

```
タイトル：[テーマ]で悩んでるママへ、保育士が伝えたいこと

1. はじめに（冒頭文）
2. よくある誤解：[テーマ]で最初に手放していいこと
3. 保育園で見てきたこと：うまくいく家・いかない家の違い
4. 家でできること：今日から試せる関わり方1つ
5. おわりに（結び文）
```

## 冒頭文テンプレート（約280文字）

```
こんにちは。ちゃむ。です。

[テーマ]で悩んで、この記事にたどり着いてくれたんですよね。夜中に検索して、読んでは閉じて、また検索して——その繰り返しの途中かもしれません。

この記事では、[テーマ]でつまずいたときに「今日から何を見ればいいか」を、園での経験と、私自身の子育てでの失敗をもとにお話しします。

私は保育士を16年、今は児童支援員をしながら小学生3人を育てています。専門家としてではなく、同じところで悩んだ母親として書きました。

少しだけ、お付き合いください。
```

## 結び文テンプレート（有料noteへの橋渡し・約190文字）

```
ここまで読んでくださって、[テーマ]で「まず何を見ればいいか」は掴めたんじゃないかなと思います。それだけで、昨日までとは違う目でお子さんを見られるはずです。

ただ——「うちの子の場合はどうなの？」というところまで踏み込みたい方もいると思います。

そういう方のために、園で見てきた実例と私の失敗談まで含めた続きを、別のnoteに書いています。必要になったときに、思い出してもらえたら嬉しいです。
```

# 第3部：有料note骨格テンプレート

## 構成（目次）

```
タイトル：保育士16年でわかった[テーマ]の本当の話

第1章：保育士の私が、我が子の[テーマ]で最初に折れた日
第2章：園で見てきた「うまくいく家」が、実はやっていなかったこと
第3章：正しい対応より先に、私が手放したもの
第4章：3人育てて分かった、「その子によって違う」の本当の意味
第5章：それでもダメな日のための、逃げ道の作り方
```

## 冒頭文テンプレート（約380文字）

```
こんにちは。ちゃむ。です。

このnoteを書いた理由を、最初に正直にお話しします。

私は保育士4年目から、[テーマ]で悩むお母さんたちの相談をたくさん受けてきました。そのたびに「大丈夫ですよ」と答えてきたけれど——自分の子育てが始まったら、その知識がまったく通じなかったんです。

園では冷静に対応できたことが、我が子相手だと感情が先に立って、何ひとつできない。「保育士なのに」という言葉が、一番自分を追い詰めました。

だからこのnoteは、知識をきれいに並べたものではありません。16年の現場で見てきたことと、3人の子育てで実際にぶつかった壁の記録です。

「頑張ってるのに、うちだけうまくいかない気がする」——そう思いながら今日も一日を終えようとしている人に、届けたくて書きました。
```

## 章タイトルのトーン（体験・本音ベース）

```
第1章：保育士の私が、我が子の[テーマ]で最初に折れた日
第2章：園で見てきた「うまくいく家」が、実はやっていなかったこと
第3章：正しい対応より先に、私が手放したもの
第4章：3人育てて分かった、「その子によって違う」の本当の意味
第5章：それでもダメな日のための、逃げ道の作り方
```

**章タイトルのルール**
- 「第○章：〜する方法」「〜のやり方」は禁止。知識集に見える
- 「折れた日」「手放したもの」「逃げ道」など感情・体験ベースにする
- 読者が「これ自分のことだ」と思える言葉を1つ入れる

## 購入後の一言（読了者へのメッセージ）

```
ここまで読んだあなたはもう、ひとりで抱えてないよ。今夜は少し早く寝てね
```

## 有料noteの価格・ボリューム目安

| 文字数 | 価格帯 | 使い方 |
|-------|-------|-------|
| 3,000〜5,000文字 | 300〜500円 | 入門・単発テーマ |
| 8,000〜15,000文字 | 800〜1,500円 | 深掘り・実例多め |

- 最初は低価格（300〜500円）で出して反応を見る
- 評価が高いものをアップグレードして月1訴求に使う（mojibake_writing式）
