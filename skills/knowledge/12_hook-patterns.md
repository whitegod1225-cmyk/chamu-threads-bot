# 12_hook-patterns.md ── バズフック集

購入者限定特典「バズフック集」をちゃむ。用にknowledge化したもの。
05_affiliate.mdの冒頭フックパターン集と併用する。

> **使い方**：カテゴリーから「今の商品・投稿の目的」に合うものを選び、
> ちゃむ。の文体（友人への語りかけ・元保育士視点）に言い換えて使う。
> 「全財産使っても〜」「人生損してる」など大げさすぎる表現はちゃむ。トーンにNG。

---

## 問題提起型
読者の「え、それって損なの？」を引き出す。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇に3万円使う人いるけど正直意味ない | 高額商品・見直しを促す系 |
| それって実は損してるって知ってた❓ | 見直し系商品全般 |
| 〇〇って実は必要ないんだよね | 常識を覆す代替商品 |
| みんな〇〇で失敗しすぎ問題 | 購入前の情報提供 |
| 〇〇選びで9割が勘違いしてること | 選び方ガイド系 |

---

## データ訴求型
数字で信頼と具体性を出す。ちゃむ。は実体験の数字が強い。

| フック例文 | 使えるシーン |
|-----------|------------|
| 2ヶ月で〇〇が変わった方法がこれ | ビフォーアフター |
| 5年使って結論出た | 長期レビュー・耐久性 |
| 1日30円で〇〇が手に入る計算 | コスパ訴求・日割り計算 |
| 3時間が15分になったやつ | 時短・効率化商品 |
| 10人中9人が〇〇を選ぶ理由 | 多数派・安心感訴求 |

---

## 共感獲得型
「わかる」「自分のことだ」と思わせる。育児ジャンルで特に効く。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇で悩んでたけど、これで変わった | 悩み解決・変化系 |
| 誰にも言えなかった〇〇の悩み、解決した | デリケートな悩み系 |
| 同じ悩みの人、マジで見て欲しい | 共感コミュニティ形成 |
| 〇〇ガチ勢の私が唯一認めた | 専門性・信頼性（保育士視点と相性◎） |
| ズボラでも続けられた唯一の方法 | 継続性が課題の商品 |

## 今すぐ型
セール・再入荷・値上げ前などに使う。タイミング命。

| フック例文 | 使えるシーン |
|-----------|------------|
| 明日値上げするらしい | 価格改定前の緊急性 |
| 在庫復活したから秒で買った | 再入荷・希少性 |
| 今日だけ40%オフとかヤバい | タイムセール・限定割引 |
| マジで今年買ってよかったNo.1 | 年末の購入後押し |
| 売り切れる前に教えとく | 品薄・希少性訴求 |

## 裏側暴露型
「業界の裏側を知ってる人」として語る。元保育士・専門家ポジションと相性◎。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇業界の人が絶対これ買わない理由 | 業界知識・内部情報 |
| 実は現場の人もこっち使ってる | プロの選択・信頼性（保育士時代の経験で語れる） |
| 広告費かけてないから安いだけ説 | コスパの理由説明 |
| メーカーが言わない〇〇の真実 | 隠れた情報・暴露系 |
| 〇〇の正しい使い方、説明書に書いてない | 裏技・応用法 |

## 鉄板リピート型
長く使ってきた実感を語る。ちゃむ。の「3児の母」経験年数が活きる。

| フック例文 | 使えるシーン |
|-----------|------------|
| 3年使って分かった本当の価値 | 長期使用の確信 |
| 結局いつもこれに戻ってくる | 定番・安定感 |
| 何回リピートしたか数えてない | 高リピート率商品 |
| 迷ったら絶対これ買っとけ | 鉄板おすすめ |
| これ持ってない人、損してるよ | 普遍的価値の訴求（「人生損してる」はNG→「損してるよ」に和らげる） |

| フック例文 | 使えるシーン |
|-----------|------------|
| フォロワーさんだけに教えたい | 限定感・特別感 |
| 本当は秘密にしたかった | 独占欲を刺激 |
| 身内にしか教えてなかったやつ | 内輪・信頼関係 |
| これ見た人だけ得する情報 | 閲覧者特権 |
| メモ必須、保存版 | 保存アクション誘導 |

## ガチ推し型
熱量を前面に出す。ちゃむ。トーンでは「布教してる」「ガチで言う」くらいが限度。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇で悩んでる人、騙されたと思って試して | 強力プッシュ・背中押し |
| 〇〇ならこれ以外ありえない | 唯一無二の推奨 |
| 友達全員に布教してる | 口コミ・紹介実績 |

> ⚠️ 「全財産使っても買い直す」「人生で1番」は大げさすぎてちゃむ。トーンにNG

## ビフォーアフター型
変化・成果を見せる。数字や期間をセットで入れると信頼感UP。

| フック例文 | 使えるシーン |
|-----------|------------|
| 半年前の自分に教えたい | 過去の自分への推奨 |
| 使う前と使った後で全然違う | 劇的な変化 |
| 〇〇しただけでこの変化 | シンプルな効果 |

## 比較検証型
「調べた・試した」ポジション。手間を代わりにやってあげた感。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇と〇〇、ガチで比較してみた | 詳細比較・公平性 |
| 安いやつと高いやつの違いがエグい | 価格差の正当性 |
| SNSで話題の〇〇、全部試した結果 | 網羅的検証 |

## お悩み解決型
悩みの言語化から入る。育児ジャンルで最もハマりやすい型。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇の悩み、3日で消えた | 即効性アピール |
| 諦めかけてた時に出会った | 救世主的存在 |
| 〇〇難民にマジで教えたい | 代替品探してる人向け |
| ずっと探してたのがやっと見つかった | 長年の探求の終着点 |
| もう〇〇で悩まなくていい | 悩み解放の約束 |

## リアル体験型
「試した人の生の声」として信頼を積む。

| フック例文 | 使えるシーン |
|-----------|------------|
| 1ヶ月ガチで使ってみた感想 | 短期集中レビュー |
| 買う前に絶対知っておくべきこと | 購入前必読情報 |
| 使い始めて3日で効果実感した | 即効性の体験 |
| 2回目買う時に気づいたこと | リピート時の新発見 |

## 失敗回避型
「失敗を防ぐ情報」として読まれやすい。保育士の専門知識と組み合わせると強い。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇買う前にこれ見て | 購入ミス防止 |
| 私が〇〇で失敗した理由 | 失敗談からの学び |
| 〇〇選びで絶対やっちゃダメなこと | NG行動の共有 |
| 損しないための〇〇の選び方 | 正しい選択ガイド |

## 新発見報告型
「知らなかった」驚きを共有する。保育士知識の「実は〇〇だった」と相性◎。

| フック例文 | 使えるシーン |
|-----------|------------|
| こんな商品あったの知らなかった | 新規性・驚き |
| バズる前に紹介しとく | 先行者利益 |
| 知名度低いけど超優秀 | 隠れた名品 |

## 今話題型
流行・SNSの拡散感を使う。バズ投稿の二次利用にも。

| フック例文 | 使えるシーン |
|-----------|------------|
| 今マジでこれがバズってる | 流行・トレンド |
| 〇〇クラスタの間で話題のやつ | コミュニティ内人気 |
| みんなこれ買ってるらしい | 社会的証明 |

## 革命宣言型
「革命起きてる」「世界変わった」で商品・収納・方法の劇的な変化を伝える。S-4（速報型）の称賛版。

| フック例文 | 使えるシーン |
|-----------|------------|
| ちょ、待って。〇〇に革命起きてる | 収納・インテリア・新商品 |
| 〇〇の世界が変わってた | カテゴリ全体の新常識 |
| 〇〇のやり方、もう古かったんだけど | 既存の方法からの転換 |
| 〇〇が進化しすぎてて追いつけない | 商品の急速な進化 |

> 「革命」は誇張フレームなので1投稿1回まで。
> スライド2は複数ブランドの画像まとめ＋ブランド名だけでOK（URL省略も可）。
> ちゃむ。トーンでは「収納に革命」より「〇〇の悩みが消えた」のほうが自然なことも。

---

## 時事脅威報告型
「今まさに起きている」緊張感で読み進めさせる。感染症・季節ネタ・育児ニュースと相性◎。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇が保育園で流行り始めたって… | 感染症・季節の病気 |
| 〇〇が今年また猛威を振るってる | 年間トレンドの脅威 |
| 厚労省が注意喚起出してるの知ってた？ | 公的機関の情報を引用 |
| 〇〇のピークが来週だって知ってた？ | 時期が迫る緊張感 |

> 保育士視点＋公的データを組み合わせると唯一無二のフックになる。
> 「流行り始めた」「今年また」など**進行形・繰り返し**の言葉で緊張感を高める。

## ミスリード→ネタバラシ型
フックで読者に意図的に誤解させ、スライド2でオチを出す。笑いと「騙された！」感を作る。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇とか炎上しそうだけど | 商品名や行動が一見問題に見える時 |
| 保育士が〇〇してるって言ったら引かれそうだけど | 保育士として意外に見える行動 |
| こんな時間に子どもに〇〇デビューさせた | 食べ物・行動のミスリード |
| 〇〇に〇〇あげちゃった | 意外な組み合わせ・年齢 |

> スライド2で「いやいや」「これ〇〇なんだけど」とツッコミを入れてオチを出す。
> 商品説明はほぼ不要。ネタバラシ自体が紹介になる。週1〜2本の笑いポスト向け。

---

## 目撃→二度見型
外出先での他人の行動を「目撃」したシーンから始める。S-2（日常シーン報告型）の外出版。

| フック例文 | 使えるシーン |
|-----------|------------|
| これこれ！電車で前に座ってた人が〇〇してて | 公共の場での発見 |
| 支援センターで来てたママが使ってて | 育児現場での目撃 |
| お迎えで見かけたんだけど、ガン見してしまった | 保育園・幼稚園での発見 |
| 街で見かけてから頭から離れない | ショッピングエリアでの目撃 |

> 目撃した「行動」（リップを塗ってた・バッグから出した）を具体的に書くと場面が浮かぶ。
> スライド1〜2は画像が主役になることが多い。テキストは最小限でOK。

---

## 問いかけ→天才称賛型
「誰が作ったの？」「何周目？」と問いかけることで商品・ブランドへの驚きと称賛を表現する。

| フック例文 | 使えるシーン |
|-----------|------------|
| 誰ですか？〇〇した人は。天才なんよ… | 独自な機能・デザインの商品 |
| 〇〇さん、今、人生何周目ですか？ | ブランドの商品群まとめ紹介 |
| 誰がこれを思いついたの、すごすぎる | 意外な発想の商品 |
| なんで私の悩みを知ってるんですか？ | ブランドへの称賛 |

> 「天才なんよ」「天才なんだもの」は短い断言で熱量を出す。
> S-11（困る・やめて）とは逆方向の称賛型。機能・デザインの独自性を紹介する時に使う。
> まとめリスト版（①〜⑩）は保存率が高いボリューム投稿として月1〜2回活用。

## ブランド擬人化ツッコミ型
「〇〇本気出しすぎ」「やめて」で商品を主語にしてツッコむ。「困る=最高」の逆説フレーム。

| フック例文 | 使えるシーン |
|-----------|------------|
| 待って、〇〇本気出しすぎ | コラボ・限定品・新作 |
| 〇〇、全部欲しくなるじゃないか | 複数商品のまとめ紹介 |
| 好きなもの全部〇〇するのやめて（無理） | ブランドコラボ・ギフトセット |
| 〇〇が本気になったらこうなるの | 新ライン・フルリニューアル |

> 「やめて」「困る」「無理」は最高すぎて困るという意味の逆説ツッコミ。ちゃむ。トーンでは「困る」「無理」のほうが自然なことも。

---

## 比較優勝宣言型
話題・有名な選択肢を「認めてから」自分の本命を宣言する。読者に結論を出してあげる型。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇も良いけど…〇〇が優勝だった | 人気商品との比較 |
| バズってるのわかるけど、うちでは〇〇が優勝してる | SNS話題商品との比較 |
| 楽天1位も見たけど、正直こっちのほうが好き | ランキング商品との比較 |
| 〇〇で話題の〇〇より、これのほうが合ってた | テレビ・雑誌掲載品との比較 |

> 「優勝」は比較を経た上での最強の断言。1投稿1回まで使う。
> 雑誌付録・SNSバズ商品など「旬の話題」への乗り方として使うと読まれやすい。

---

## 炎上覚悟共感型
「言いにくいけど言う」覚悟宣言で注意を引き、専門家や当事者としての誠実さを示す。

| フック例文 | 使えるシーン |
|-----------|------------|
| 炎上覚悟だけど、〇〇が大事だよ | 育児の常識を覆す本音 |
| 言いにくいけど、現場では当たり前のことを言う | 保育士視点の本音 |
| 保育士から正直に言います | 専門家の覚悟ある発言 |
| みんなが言わないことを言う | タブーへの切り込み |

> スライド1で読者の「頑張り」を全力で認めてから転換する。批判ではなく寄り添いとして機能する。
> 「炎上覚悟」は誠実さの表明。ちゃむ。トーンでは「正直に言います」くらいに和らげてもOK。

---

## 専門家SOSサイン型
問題を「意志や努力の問題」ではなく「心のサイン」として再定義する。

| フック例文 | 使えるシーン |
|-----------|------------|
| 危険です。〇〇は〇〇のSOSサインです | 怒り・疲れ・無気力の本質 |
| これ、ただの甘えじゃないよ。心のサインだよ | 読者の自己否定に応える |
| 〇〇が続くのは、〇〇が足りてないサインかも | 専門的な観察を提示 |
| 〇〇が抑えられないのは、あなたのせいじゃない | 免責＋専門解説 |

> 直後に「真の原因」を提示してから解決策に進む。
> 「頑張り方を変えるのではなく、頑張る量を減らす」のように既存の努力を否定しないことが重要。

---

## 著名人文脈乗り型
話題の人物・番組のキーワードを入り口にして読者の注意を引く。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇が話してた「〇〇」が楽しすぎる | 著名人・番組の話題に乗る |
| 〇〇のやつ試してみたら想像以上だった | 話題の方法を実践報告 |
| 最近よく聞く「〇〇」、実は科学的に理にかなってる | トレンドを専門知識で解説 |

> 著名人の名前の代わりに「フォロワーさんから教えてもらった」「支援センターで聞いた」でも機能する。
> 「実は科学的に理にかなってる」というフレームは保育士・専門家ポジションと相性◎。

---

## 速報興奮型
「今まさに発見した」リアルタイム感で読者を引き込む。限定・再入荷・セール開始に使う。

| フック例文 | 使えるシーン |
|-----------|------------|
| え、待って待って！〇〇に〇〇が追加されてる！ | 再入荷・限定品・アウトレット |
| これ知ってた？さっき見つけたんだけど | 新発見の速報 |
| 気づいたら復活してた！これは早めに見て | 再入荷の速報 |
| セール始まったの今知って急いでる | セール開始の興奮 |

> フックの感嘆符・繰り返し表現（「待って待って」）で発信者の興奮を可視化する。
> ちゃむ。トーンでは感嘆符は1〜2個まで。「ーーッッ」の連打は控えめに。

---

## 逆説後悔型
ネガティブに見える出だしで引き込み、スライド2で「実は良い意味」と転換する。

| フック例文 | 使えるシーン |
|-----------|------------|
| 本当に後悔してます…（いい意味で） | 予想以上だった商品 |
| やめとけばよかった…子どもが強くなりすぎて | 成長・変化系 |
| 買わなきゃよかった…追いつけなくなって | 子どもの成長加速系 |
| 正直後悔してる。なんでもっと早く買わなかったんだろ | 購入が遅れた後悔 |

> スライド1でネガティブに見せて「続きが気になる」状態を作り、スライド2で転換する。
> 効果を連鎖で積み重ねていく（体力→体幹→姿勢→感情安定…）と読者が引き込まれる。

## 衝動購入報告型
「見つけた瞬間に買った」という行動の事実から始める。

| フック例文 | 使えるシーン |
|-----------|------------|
| 見つけてすぐ買った | 価格・内容のコスパが強い商品 |
| 気づいたらカートに入れてた | 衝動感を演出 |
| 見た瞬間これだと思って即注文した | 直感購買 |
| この値段で〇〇は反則 | 価格とクオリティのギャップ |

> 「なぜ買ったか」をフックの直後に1行で添える（価格・内容・子どもへの確信）。
> ちゃむ。トーンでは「反則」「やばい」は1投稿に1回まで。

---

## 笑えるエピソード導入型
笑えるエピソードがつかみになり、商品説明への自然な橋渡しになる。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇が口癖で、冗談で〇〇を提案したら… | 家族・友人の意外な反応 |
| 子どもに冗談で聞いたら本気で答えてきて | 子どもの反応の面白さ |
| 夫に「これどう？」って聞いたら予想外の返事が | 夫・パートナーの意外な反応 |
| 保育園の子に言ったら全員で真剣に悩み始めた | 子どもの純粋な反応 |

> 笑いで読者の心を柔らかくしてから商品に入る。押しつけ感ゼロ。
> 保育士時代の子どものエピソードと相性◎。

---

## 日常シーン報告型
「昨日〜した」で始める。実体験の場面報告がフックになる。

| フック例文 | 使えるシーン |
|-----------|------------|
| 昨日、ママ友の家に行ったら〇〇使っててさ | 第三者宅・訪問シーン |
| 先日支援センターで見かけたんだけど | 外出先での発見 |
| 保育園の先生が使ってるの見て気になった | 現場・専門家の使用 |
| 子どもの友達が持ってきてて羨ましかった | 子ども経由の発見 |

> 第三者が使っている場面で商品を「登場」させることで、押しつけ感がなくなる。
> スライド2で「使ってみたら〇〇だった」と転換すると自然な流れになる。

価格破壊・お得感を前面に出す。節約スタイルのちゃむ。と相性◎。

|-----------|------------|
| この値段でこのクオリティはバグ | 価格破壊・お得感 |

## 本音・驚き・つかみワード型
書き出し1行だけで「続きが気になる」状態を作るフレーズ。どのカテゴリにも組み合わせて使える。

| フック書き出し | 機能・使い方 |
|-------------|------------|
| 黙ってましたが、 | 隠してた情報を打ち明けるトーン。秘密の共有感 |
| ぶっちゃけると、 | 本音・実情を語るトーン。共感と信頼を同時に作る |
| 1億回言ってますが.. | 繰り返し強調。「これは本当に大事」という熱量を示す |
| 絶対に覚えておいて.. | 「重要情報を伝える」宣言。育児知識・選び方と相性◎ |
| もう辞めます。 | 意外性で止まらせる。「〇〇するのをやめます」に続く |
| 〇〇がヤバすぎる.. | シンプルな衝撃表現。商品・発見・価格に使う |
| ガチですか、、、 | 驚き・困惑を表現。「これが〇〇円って本当？」という感情 |
| えっ、知らないんですか.. | 知識格差をやさしく突く。保育士知識・隠れた名品に最適 |
| 9割の人が.. | 多数派との逆張り。「〇〇してること知らない」と続ける |

> ちゃむ。での使い方：「えっ、知らないんですか..」「ぶっちゃけると、」「黙ってましたが、」は友人への語りかけトーンと相性◎。
> 「もう辞めます。」は次の行で「〇〇を使い続けるのをやめます」と明かすオチが必要。
> 「ヤバすぎる」「ガチ」は1投稿1回まで。

## 定番否定・ニュース共感型（iro_to_kurahsiスタイル）
有名ブランド・定番の選択肢を「謝罪」や「危なかった」で否定し、代替品に誘導する。短くてPR感がない。

| フック書き出し | 機能・使い方 |
|-------------|------------|
| ごめん。〇〇で〇〇買うのやめた。 | 定番ブランドへの謝罪＋乗り換え宣言。「だってさ、」で続けて理由を出す |
| あぁ…危ない。〇〇行く前にここ知って良かった。 | 定番の選択肢を「危なかった」と笑いで回避した安堵感 |
| 〇〇値上げって辛過ぎ…さすがにいつかは…と思ってたけど | 値上げ・廃番ニュースへの共感→スライド2で「今が狙い目」に転換 |
| 感度高い子は知ってる… | 情報格差フック。「まだ知らない人もいる希少情報」の演出 |
| えっ！今年の〇〇これにしよ！ | 見た瞬間の即決宣言。S-8より短くシンプル |

> ちゃむ。への注意：
> - 「ごめん。ユニクロやめた」は西松屋・しまむら・無印など育児ジャンルの定番に置き換えられる
> - 「感度高い子は知ってる」は若い向けのフレームなのでちゃむ。では「知る人ぞ知る」「実は知る人には有名な」に言い換えた方が自然
> - 値上げ系フックはS-19（ネガティブニュース→緊急性転換型）と組み合わせて使う

## 感情爆発・友人報告型（mayo_mama20スタイル）
「怒り・信じられない・後悔」など感情をそのまま1行で出し、スライド2でオチを作る。短くてPR感がない。

| フック書き出し | 機能・使い方 |
|-------------|------------|
| コレは〇〇していいよね？だってさ、 | 感情の問いかけ→スライド2で「可愛すぎて怒ってる」に転換 |
| は？〇〇が〇〇？？？ | 速報×驚き。「無料」「半額」「復活」など信じられない情報に |
| なぜ〇〇の時に見つけられなかったの… | タイムラグの後悔。「上の子の時に知りたかった」系の共感 |
| つまり〇〇すれば〜ごにょごにょ… | 読者と一緒に考える共犯感。言いかけで余白を作る |

> ちゃむ。への注意：
> - 「は？」「だってさ、」はちゃむ。トーンに合う口語感
> - 絵文字連打（😭😭😭）はちゃむ。では2〜3個まで
> - 「ごにょごにょ」は楽天セール系の投稿で使うと自然

## ちゃむ。への適用メモ

**特に相性が良いカテゴリ**
- **裏側暴露型**・**失敗回避型**・**新発見報告型**：元保育士の専門知識と組み合わせると唯一無二のフックになる
- **共感獲得型**・**お悩み解決型**：育児の悩みを代弁する形でそのまま使える
- **データ訴求型**：「3年使って」「3人で使い倒して」など自分の体験数字に置き換えると強い

**NGになりやすい表現（そのまま使わない）**
- 「人生損してる」→「知らないと損するよ」くらいに和らげる
- 「全財産使っても」「人生で1番」→大げさすぎ・AI感が出る
- 「〇〇界に革命起きてる」→広告感が強すぎ
