# 12_hook-patterns.md ── バズフック集

購入者限定特典「バズフック集」をちゃむ。用にknowledge化したもの。
05_affiliate.mdの冒頭フックパターン集と併用する。

> **使い方**：カテゴリーから「今の商品・投稿の目的」に合うものを選び、
> ちゃむ。の文体（友人への語りかけ・元保育士視点）に言い換えて使う。
> 「全財産使っても〜」「人生損してる」など大げさすぎる表現はちゃむ。トーンにNG。

---

## 問題提起型
読者の「え、それって損なの？」を引き出す。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇に3万円使う人いるけど正直意味ない | 高額商品・見直しを促す系 |
| それって実は損してるって知ってた❓ | 見直し系商品全般 |
| 〇〇って実は必要ないんだよね | 常識を覆す代替商品 |
| みんな〇〇で失敗しすぎ問題 | 購入前の情報提供 |
| 〇〇選びで9割が勘違いしてること | 選び方ガイド系 |

---

## データ訴求型
数字で信頼と具体性を出す。ちゃむ。は実体験の数字が強い。

| フック例文 | 使えるシーン |
|-----------|------------|
| 2ヶ月で〇〇が変わった方法がこれ | ビフォーアフター |
| 5年使って結論出た | 長期レビュー・耐久性 |
| 1日30円で〇〇が手に入る計算 | コスパ訴求・日割り計算 |
| 3時間が15分になったやつ | 時短・効率化商品 |
| 10人中9人が〇〇を選ぶ理由 | 多数派・安心感訴求 |

---

## 共感獲得型
「わかる」「自分のことだ」と思わせる。育児ジャンルで特に効く。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇で悩んでたけど、これで変わった | 悩み解決・変化系 |
| 誰にも言えなかった〇〇の悩み、解決した | デリケートな悩み系 |
| 同じ悩みの人、マジで見て欲しい | 共感コミュニティ形成 |
| 〇〇ガチ勢の私が唯一認めた | 専門性・信頼性（保育士視点と相性◎） |
| ズボラでも続けられた唯一の方法 | 継続性が課題の商品 |

---

## 今すぐ型
セール・再入荷・値上げ前などに使う。タイミング命。

| フック例文 | 使えるシーン |
|-----------|------------|
| 明日値上げするらしい | 価格改定前の緊急性 |
| 在庫復活したから秒で買った | 再入荷・希少性 |
| 今日だけ40%オフとかヤバい | タイムセール・限定割引 |
| マジで今年買ってよかったNo.1 | 年末の購入後押し |
| 売り切れる前に教えとく | 品薄・希少性訴求 |

---

## 裏側暴露型
「業界の裏側を知ってる人」として語る。元保育士・専門家ポジションと相性◎。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇業界の人が絶対これ買わない理由 | 業界知識・内部情報 |
| 実は現場の人もこっち使ってる | プロの選択・信頼性（保育士時代の経験で語れる） |
| 広告費かけてないから安いだけ説 | コスパの理由説明 |
| メーカーが言わない〇〇の真実 | 隠れた情報・暴露系 |
| 〇〇の正しい使い方、説明書に書いてない | 裏技・応用法 |

---

## 鉄板リピート型
長く使ってきた実感を語る。ちゃむ。の「3児の母」経験年数が活きる。

| フック例文 | 使えるシーン |
|-----------|------------|
| 3年使って分かった本当の価値 | 長期使用の確信 |
| 結局いつもこれに戻ってくる | 定番・安定感 |
| 何回リピートしたか数えてない | 高リピート率商品 |
| 迷ったら絶対これ買っとけ | 鉄板おすすめ |
| これ持ってない人、損してるよ | 普遍的価値の訴求（「人生損してる」はNG→「損してるよ」に和らげる） |

## 特別公開型
「あなただけに教える」感を出す。フォロワー限定感との相性◎。

| フック例文 | 使えるシーン |
|-----------|------------|
| フォロワーさんだけに教えたい | 限定感・特別感 |
| 本当は秘密にしたかった | 独占欲を刺激 |
| 身内にしか教えてなかったやつ | 内輪・信頼関係 |
| これ見た人だけ得する情報 | 閲覧者特権 |
| メモ必須、保存版 | 保存アクション誘導 |

## ガチ推し型
熱量を前面に出す。ちゃむ。トーンでは「布教してる」「ガチで言う」くらいが限度。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇で悩んでる人、騙されたと思って試して | 強力プッシュ・背中押し |
| 〇〇ならこれ以外ありえない | 唯一無二の推奨 |
| 友達全員に布教してる | 口コミ・紹介実績 |

> ⚠️ 「全財産使っても買い直す」「人生で1番」は大げさすぎてちゃむ。トーンにNG

## ビフォーアフター型
変化・成果を見せる。数字や期間をセットで入れると信頼感UP。

| フック例文 | 使えるシーン |
|-----------|------------|
| 半年前の自分に教えたい | 過去の自分への推奨 |
| 使う前と使った後で全然違う | 劇的な変化 |
| 〇〇しただけでこの変化 | シンプルな効果 |

---

## 比較検証型
「調べた・試した」ポジション。手間を代わりにやってあげた感。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇と〇〇、ガチで比較してみた | 詳細比較・公平性 |
| 安いやつと高いやつの違いがエグい | 価格差の正当性 |
| SNSで話題の〇〇、全部試した結果 | 網羅的検証 |

## お悩み解決型
悩みの言語化から入る。育児ジャンルで最もハマりやすい型。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇の悩み、3日で消えた | 即効性アピール |
| 諦めかけてた時に出会った | 救世主的存在 |
| 〇〇難民にマジで教えたい | 代替品探してる人向け |
| ずっと探してたのがやっと見つかった | 長年の探求の終着点 |
| もう〇〇で悩まなくていい | 悩み解放の約束 |

## リアル体験型
「試した人の生の声」として信頼を積む。

| フック例文 | 使えるシーン |
|-----------|------------|
| 1ヶ月ガチで使ってみた感想 | 短期集中レビュー |
| 買う前に絶対知っておくべきこと | 購入前必読情報 |
| 使い始めて3日で効果実感した | 即効性の体験 |

## 失敗回避型
「失敗を防ぐ情報」として読まれやすい。保育士の専門知識と組み合わせると強い。

| フック例文 | 使えるシーン |
|-----------|------------|
| 〇〇買う前にこれ見て | 購入ミス防止 |
| 私が〇〇で失敗した理由 | 失敗談からの学び |
| 〇〇選びで絶対やっちゃダメなこと | NG行動の共有 |
| 損しないための〇〇の選び方 | 正しい選択ガイド |

---

## 新発見報告型
「知らなかった」驚きを共有する。保育士知識の「実は〇〇だった」と相性◎。

| フック例文 | 使えるシーン |
|-----------|------------|
| こんな商品あったの知らなかった | 新規性・驚き |
| バズる前に紹介しとく | 先行者利益 |
| 知名度低いけど超優秀 | 隠れた名品 |

| フック例文 | 使えるシーン |
|-----------|------------|
| 今マジでこれがバズってる | 流行・トレンド |
| 〇〇クラスタの間で話題のやつ | コミュニティ内人気 |

---

## コスパ神型
価格破壊・お得感を前面に出す。節約スタイルのちゃむ。と相性◎。

| フック例文 | 使えるシーン |
|-----------|------------|
| この値段でこのクオリティはバグ | 価格破壊・お得感 |
| 1000円以下で買える神アイテム | 超プチプラ訴求 |
| 高いやつと品質変わらない件 | コスパ最強証明 |
| ここまで安くていいの❓レベル | 罪悪感を伴う安さ |

---

## ちゃむ。への適用メモ

**特に相性が良いカテゴリ**
- **裏側暴露型**・**失敗回避型**・**新発見報告型**：元保育士の専門知識と組み合わせると唯一無二のフックになる
- **共感獲得型**・**お悩み解決型**：育児の悩みを代弁する形でそのまま使える
- **データ訴求型**：「3年使って」「3人で使い倒して」など自分の体験数字に置き換えると強い

**NGになりやすい表現（そのまま使わない）**
- 「人生損してる」→「知らないと損するよ」くらいに和らげる
- 「全財産使っても」「人生で1番」→大げさすぎ・AI感が出る
- 「〇〇界に革命起きてる」→広告感が強すぎ
