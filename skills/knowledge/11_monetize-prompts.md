# 11_monetize-prompts.md ── マネタイズ・成長プロンプト集

## 概要
フォロワー獲得・信頼構築・導線設計に特化した12プロンプト。
`/writer` が投稿を磨く時、`/analyst` がアカウント戦略を立てる時に使う。

`[  ]` の部分に対象の投稿文・エピソード・過去投稿などを入れて使う。

---

## 滞在時間・離脱防止

### 1. 離脱防止
読者が読むのをやめる箇所を特定し、続きへの仕掛けを埋め込む。

```
この投稿で読者が読むのをやめる箇所を全部特定して、
続きが気になる仕掛けを1つずつ埋め込んで。
```

### 5. シリーズ設計
1投稿バズ依存から抜け出す連続読み設計。

```
このネタをフォロワーが次も読みたくなる5投稿のシリーズに分解して、
各回の末尾に続きへの橋渡しを設計して。
```

---

## 読者心理・共感設計

### 2. 欲求剥き出し
読者が口に出さない本音の欲求を掘り出す。

```
読者が絶対に口に出さないけど心で思っている本音の欲求を3つ推測して、
それぞれに刺さるフック1行を作って。
```

### 4. 反論先回り
「自分には関係ない」という言い訳を本文内で先に潰す。

```
この投稿を読んだ人が"自分には関係ない"と思う理由を4つ予測して、
それを本文の中で先に潰す構成に書き直して。
```

### 6. 未来日記型
1ヶ月後のゴールから逆算して今日の投稿を設計する。

```
私をフォローして1ヶ月後の読者が友人に送るLINEを書いて、
そのLINEが自然に届くよう今日の投稿を逆算して構成して。
```

---

## 信頼構築・差別化

### 3. 競合差別化
同ジャンルの発信者が言わない不都合な真実を、炎上ではなく信頼獲得のトーンで出す。

```
同ジャンルの発信者が絶対に言わない不都合な真実を1つ暴露する投稿を、
炎上ではなく信頼獲得になるトーンで書いて。
```

### 7. 信頼蓄積
実績ゼロ時代の惨めなエピソードを共感とフォロー欲に変換する。

```
私の実績ゼロ時代の惨めで具体的なエピソードを1つ入力するから、
最大限の共感とフォロー欲に変換する投稿を書いて。

[エピソード：　]
```

### 11. 同一人物感設計
「この人だけが自分を分かってくれる」と感じさせる共通の敵・価値観・言語を仕込む。

```
この投稿を読んだ人が"この人だけが自分のことを分かってくれる"と感じるよう、
共通の敵・共通の価値観・共通の言語を仕込んで書き直して。
```

---

## アカウント設計・導線

### 8. ペルソナ転写
過去投稿から言葉の癖・リズム・思考パターンを抽出し、以後の文体に固定する。

```
私の過去投稿3つを読み込んで言葉の癖・リズム・思考パターンを抽出して、
今後はそのパターンで書くよう自分をアップデートして。

[過去投稿①：　]
[過去投稿②：　]
[過去投稿③：　]
```

### 9. フォロー導線設計
投稿→プロフ→フォロー の3ステップで離脱しない導線を設計する。

```
投稿→プロフ→フォロー の3ステップで離脱しない導線を設計して、
各ステップで読者が感じる不安も一緒に潰して。
```

### 10. 離脱ユーザー奪還
一度フォローを外した人が戻りたくなる投稿パターンを、離れた理由ごとに作る。

```
一度フォローを外した人が戻りたくなる投稿パターンを、
離れた理由ごとに3つ作って。
```

### 12. 初見フォロー設計
初めて来た人が3秒でフォローするプロフ文と初投稿を同時に設計する。

```
私のアカウントを初めて見た人が3秒でフォローボタンを押したくなるプロフ文と、
それと世界観を統一した初投稿の文章を同時に作って。
```
