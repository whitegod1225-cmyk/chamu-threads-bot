# 05_affiliate.md ── アフィリエイト方針
## 基本方針
- 楽天アフィリエイトをメインに使用
- 「売りたい」感を出さない。実体験ベースで自然に紹介

## 商品カテゴリ（優先順）
1. ベビー用品（おむつ・授乳グッズ・ベビー食器など）
2. 育児グッズ（抱っこ紐・ベビーカー・おもちゃ）
3. 育児と関連できる日用雑貨（収納・洗濯グッズ）
4. 料理雑貨（時短調理グッズ・離乳食関連）

## 投稿パターン
### パターン①：4行実用レビュー型（メイン）
- 1行目：インパクトある一文
- 2行目：どんな悩みを解決するか
- 3行目：使ってみた率直な感想
- 4行目：自然なリンク誘導

### パターン②：短文インパクト型（たまに）
- 1〜2行のみの強い断言
- リンクをそのまま置く

## 注意点
- 「おすすめです！」「ぜひ！」連発はNG
- 一つの投稿に複数商品を詰め込まない

## note導線設計

### 基本導線
Threads投稿 → 無料note（集客装置）→ 有料note（購入）

直接有料noteへ誘導する「直通ルート」はNG。信頼がない状態では購入されない。

### 無料noteの役割
- 購入ハードルをゼロにする（タダだから読まれる → 価値を感じる → 続きが欲しくなる）
- 信頼を先払いする（読後に「この人は本物だ」と感じさせる）
- ターゲットを自動選別する（本気で興味がある人だけが無料noteまで読む）

### 無料/有料の境界線3基準
**基準1：問題の理解は無料、問題の解決は有料**
- 無料note：「なぜ自分はうまくいかないのか」を気づかせる内容
- 有料note：「どうすればうまくいくのか」の具体的な手順・解決策

**基準2：読者の言葉は無料、自分の言葉は有料**
- 無料note：読者が日常で使う言葉・感じる感情・経験する状況で語る
- 有料note：ちゃむ。独自の視点・保育士16年でしか語れない知識・オリジナルの言語化

**基準3：1テーマを深く・800〜1,200文字**
- 無料noteで扱う問題は必ず1つに絞る
- 浅く広く書いた無料noteは「なるほど感」を生まない
- 深く狭く書いた無料noteが「この人は本物だ」という信頼を生む

### 無料noteの設計手順
1. 有料noteが解決する「最終的な問題」を1行で書く
2. その最終問題の「前段階にある問題」を抽出する（これが無料noteのテーマ）
3. 前段階の問題を読者の言葉で深く掘り下げる
4. 解決策は書かない
5. 末尾に7ステップCTAを置く（→ 11_monetize-prompts.md のプロンプト13を使う）
