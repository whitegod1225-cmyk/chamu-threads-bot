# 07_ng-rules.md ── NGルール
## 絶対NG（内容面）
- 過度に暴力的・攻撃的な表現
- 売り気が見える表現（「今すぐ買って！」など）
- 根拠のない断言（「絶対に〜」「必ず〜」）
- 「シンママ」という表現（ターゲット外・対象読者は両親がいる保育園ママ）

## 絶対NG（文体面）
- 同じ語尾を連続で3回以上使わない
- AIくさい表現（「〜と言えるでしょう」「〜ではないでしょうか」など）
